If you have a Platinum or Diamond-level subscription, you can use the audit log to view changes in your KSAT console made by other admins, KnowBe4 agents, or users with certain Security Role permissions. In the audit log, you can view what changes occurred, which user made the changes, and where the changes happened up to 180 days.
\n
Accessing the Audit Log
\n
To access the audit log, follow the steps below.
\n
    \n
  1. Log in to your KSAT console.
    2. \n
  2. Select the email address in the top-right corner of the page. Then, click Audit Log.\n
    \n
    3. \n
\n
You\u2019ll be redirected to the Audit Log page.
\n
\n
Using the Audit Log
\n
To search the audit log, select the Filters drop-down menu. For more information, see the screenshot and description below.
\n
\n
    \n
  1. \nTimeframe: Select a timeframe of Last 7 days, Last 30 days, Last 90 days, or Custom. The Custom timeframe includes any time up to the last 180 days.
    2. \n
  2. \nActivities: Select one or more activities.
    3. \n
  3. \nAccount Admins: Select one or more account admins.
    4. \n
  4. \nAPI Events Only: Turn on this toggle to display only events generated by the API.
    5. \n
  5. \nProduct Sections: Select one or more sections.
    6. \n
  6. \nProduct Subsections: Select one or more subsections. You can search any subsection except Account, Phish Alert Setting, and Phish Alert Language.
    7. \n
  7. \nItems: To search this field, select one Product Subsection. Then, enter a minimum of two characters to search and select any items.
    8. \n
  8. \nChanges: To search this field, select a timeframe and select one or two Product Subsections.
    9. \n
\n
To view the exact changes that occurred, click the arrow icon.
\n
\n
\nNote:The audit log won\u2019t track changes made to sensitive information, such as passwords and API keys.
\n
\n
\n
Click Generate CSV to download a CSV file of your audit log.
\n
\n
\nImportant: CSV file downloads can\u2019t exceed 1000 rows.
\n
\n
