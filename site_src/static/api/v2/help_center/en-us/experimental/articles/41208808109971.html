The SecurityTips subtab is where you manage SecurityTip content and notification templates used in real-time coaching campaigns. SecurityTips can be used to coach your users about their risky activity and how to avoid it in the future.
\n
You can leverage preexisting SecurityTip content or upload custom graphics. SecurityTips are automatically localized to match the recipient's training language. For instructions on uploading custom graphics, see our Upload Custom SecurityTip Graphics article.
\n
  
\n
Each SecurityTip content item is represented by a card on the Media Catalog page. SecurityTips are available in image, GIF, or video formats. A GIF icon indicates the GIF format, a graphic icon indicates the graphic format, and a video icon indicates the video format. Clicking a card allows you to preview the SecurityTip content.
\n
\n
\nNote:When available, localized versions of SecurityTip content are displayed in the user\u2019s training language.
\n
\n
Media Catalog
\n
The Media Catalog page includes several interactive elements for filtering and managing SecurityTip content:
\n
  
\n
    \n
  1. \nMedia Type: Click this drop-down menu to filter media by type.
    2. \n
  2. \nSource: Click this drop-down menu to choose between System media and Custom uploaded media.
    \n
    \n
    \nNote:Only custom images are currently supported for SecurityTips.
    \n
    \n
    3. \n
  3. \nTopics:\u00a0Click this drop-down menu to filter media by category.
    4. \n
  4. \nLanguages: Click this drop-down menu to filter media by language.
    5. \n
  5. \nSearch: Enter a keyword used in the media title to filter your results.
    6. \n
  6. \nMedia Type label: This label indicates whether the media is available in image, GIF, or video format.
    7. \n
  7. \nTitle label: This label displays the title of the SecurityTip media.
    8. \n
  8. \nLanguage label: This label displays the number of available languages for the SecurityTip media.
    9. \n
  9. \nTopics label: This label displays the category for the SecurityTip media. We offer SecurityTip media for several categories, such as Malware, Social Media, and Ransomware.
    10. \n
  10. \nCreate Campaign: Click this button to create a campaign using this SecurityTip media.
    11. \n
  11. \n+ New SecurityTip: Click this button to create a custom SecurityTip.
    12. \n
\n
Notification Templates
\n
  
\n
The Notification Templates page allows you to manage and view notification templates. For information on customizing templates, see our Customize SecurityTip Notification Templates article.
