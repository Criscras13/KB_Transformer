\n
\nNote:We\u2019ve added advanced features to phishing templates! All accounts will be opted in to phishing template advanced features in 2025. For more information, see our Phishing Templates Advanced Features Guide.
\n
\n
In your KSAT console, you can use data breach phishing templates to spear-phish users who have been part of specific data breaches. These phishing templates are intended to be used along with the results of KnowBe4's Email Exposure Check Pro (EEC Pro) tool, but they can be used in any type of phishing campaign.
\n
We offer over 100 data breach phishing templates, each spoofing a specific organization that has experienced a large-scale data breach incident. The templates prompt the user to click a link and include a corresponding data entry landing page. Data entry landing pages allow you to test users\u2019 likelihood of clicking a link and entering sensitive information.
\n
\n
\nNote:Our landing pages will never save any of the entered data. We only track whether the user entered data, not the data itself.
\n
\n
How to Preview Data Breach Templates and Landing Pages
\n
To preview our data breach phishing templates, follow the steps below:
\n
    \n
  1. After signing in to your KSAT account, navigate to Phishing > Phishing Templates > System Templates.
    2. \n
  2. Locate the System Topics menu on the left side of the screen.
    3. \n
  3. Then, locate and click the Data Breach topic.
    4. \n
  4. Click the eyeball icon under the Actions column to preview the template.
    5. \n
\n
To preview our data breach landing pages, follow the steps below:
\n
    \n
  1. After signing in to your KSAT account, navigate to Phishing > Landing Pages > System Landing Pages.
    2. \n
  2. Locate the System Topics menu on the left side of the screen.
    3. \n
  3. Then, locate and click the Data Breach Phishing topic.
    4. \n
  4. Click the eyeball icon under the Actions column to preview the landing page.
    5. \n
\n
How to Create a Data Breach Phishing Campaign
\n
Follow the steps below to create a data breach phishing campaign:
\n
    \n
  1. \nRun the EEC Pro tool if you've never run it before. Existing customers with Gold or higher subscription levels will get this report automatically on a regular basis and can start with step two. For more information, see the Email Exposure Check Pro (EEC Pro) Product Manual.\n
    2. \n
  2. Review your EEC Pro results and determine which users have been involved in data breaches.\n
    \n
    \nNote: The High and Very High risk categories indicate that the user's information was part of a data breach incident.
    \n
    \n
    3. \n
  3. Group your users based on the results. For example, you may create a group called \"Kickstarter Data Breach\" and place users who were part of the Kickstarter data breach within that group.
    4. \n
  4. Set up a phishing campaign with the following settings:\n
      
    \n
    5. \n
      \n
    1. \nSend to: Select any groups you created in step three.
      2. \n
    2. \nTemplate Topics: Select the Data Breach topic. Then, you can select a template that spoofs the organization with a data breach that impacted your users. Note that a specific template must be chosen.
      3. \n
    3. \nFrequency: Select One-time.
      4. \n
    4. \nSending Period: Select Send all emails when the campaign starts.
      5. \n
    5. \nTrack Activity: We recommend setting this to 3 days.
      6. \n
    \n
  5. Leave the remaining options on their default setting.
    6. \n
  6. Click Create Campaign.
    7. \n
\n
Once the test is complete, you can review the results. You can optionally share the results with the individual users who failed your phishing test and use this opportunity to discuss the impact of data breaches. You can also enroll users who failed in a remedial training campaign using content that covers data breaches.
