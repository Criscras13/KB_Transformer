\n
Although there is typically not an interface for SpamAssassin,\u00a0you can edit your custom SpamAssassin config file to add rules allowing KnowBe4's emails through\u2013based on the header and IP address of the phishing tests.
\n
The instructions below are for a third-party software. If you run into issues whitelisting KnowBe4 in SpamAssassin,\u00a0we recommend reaching out to SpamAssassin for specific instructions. You can also\u00a0contact our support team\u00a0whenever you need assistance.
\n
