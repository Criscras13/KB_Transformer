If your USB drive test is recording the number of Opens, but not the number of Macros Enabled, check the WebClient service properties on your machine.
\n
WebClient is the Windows service that the Macros use to connect to the KnowBe4 servers to record if the Macro has been enabled. If you see that the Startup type is Disabled, the Macros will not be recorded when they run.
\n
To resolve this, set the Startup type for the WebClient service to Manual. You can also use Group Policy to set the Startup type to Manual for all of the computers in your environment.
