After you integrate your KCM GRC platform with KnowBe4's Security Awareness Training (KSAT) console, you can create KCM GRC tasks to automate evidence collection from KSAT. Then, your users' training completions will automatically be used as evidence for the tasks.\u00a0
\n
To learn how to create tasks for KSAT and work with integrated tasks, see the sections below. For general information about the KSAT integration, see our How to Integrate KnowBe4's KSAT Console with KCM GRC article.
\n
\n
Creating Integrated Tasks
\n
To create a KCM GRC task that collects evidence from KSAT, follow the steps below:
\n
    \n
  1. Log in to your KCM GRC platform.
    2. \n
  2. From your navigation panel, select the Controls tab.
    3. \n
  3. In the Name column, select the control that you would like to create a task for.
    4. \n
  4. On the View Control page that opens, scroll down to the Recurring Tasks subtab.
    5. \n
  5. Click the Create Task Schedule\u00a0button. \n
    6. \n
  6. Next to Integration, select KnowBe4 KSAT Console.
    7. \n
  7. Fill out the rest of the fields on the\u00a0Create Task pop-up window. For more information about these fields, see the screenshot and list below:\n
      \n
    1. \nTask Name: By default, the control name will display in the Task Name field. You can customize the name by entering up to 255 characters in this field.
      2. \n
    2. \nDescription: By default, the control description will display in the Description field. You can customize the description by entering up to 10,000 characters in this field.
      3. \n
    3. \nKSAT Training Campaign: In this field, select the KSAT training campaigns that you would like to collect evidence for. For more information about training campaigns, see our Creating and Managing Training Campaigns article.
      4. \n
    4. \nWhen would you like evidence to be collected?: From this field, select the time frame that you would like to collect evidence during. For more information about these options, see the list below:\n
        \n
      • \nSpecific Date: Select this option to generate evidence once, on a specific date. In the Date field that displays, enter or select a date to collect evidence on.
        • \n
      • \nRecurring Basis: Select this option to continually generate evidence based on a schedule. From the Task Frequency field that displays, select how often to generate evidence. In the Starting On field that displays, enter or select the date that evidence will start being generated on. For information about the start dates of recurring tasks, see the Recurring Task Frequencies section of our How to Work with Tasks for Controls article.
        • \n
      \n
      5. \n
    5. \nApproving Manager: Select the name of the user responsible for approving the task evidence before the task can be closed. This user will receive an email notification when this task is marked as Complete. For more information about approving tasks, see our How to Monitor and Approve Tasks article.\n
      6. \n
    6. \nSecondary Approving Manager: Select the name of the user responsible for approving the task evidence after the Approving Manager. This user will receive an email notification after the Approving Manager approves the task. For more information about approving tasks, see our How to Monitor and Approve Tasks article.\n
      7. \n
    \n\n
    8. \n
  8. Click the Create button.
    9. \n
\n
After you create an integrated task, the evidence will automatically generate during the selected time frame. If you would like, you also have the option to generate evidence instantly. After your evidence generates, you can view the evidence. For more information, see the sections below.
\n
\n
Generating Evidence Instantly
\n
If you've created an integrated task, you can generate evidence for our KSAT training campaigns instantly. For example, this feature would be helpful if you created a recurring task that happens once a year, but you unexpectedly need to generate evidence early.
\n
To generate evidence instantly, navigate to the Supporting Evidence section of the task's\u00a0View Task page. Then,\u00a0click the Generate Evidence button.
\n
\n
Viewing Evidence CSV Files
\n
After the evidence is generated, a CSV file will display in the Supporting Evidence section of the\u00a0View Task page. This CSV file contains information about the KSAT training campaigns included in the task. The rows in the CSV file contain data for each user who is enrolled in the campaign.
\n
