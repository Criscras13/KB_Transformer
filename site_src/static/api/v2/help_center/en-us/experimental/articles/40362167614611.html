From the Reports tab of your KnowBe4 account, you can view your Risk Score Activity report to monitor your organization's Risk Score-related data. The Risk Score Activity report allows you to view more details about your Risk Score.
\n
\n
\nNote:Data from your KSAT console can take up to 24 hours to process and display in your Risk Score Activity reports.
\n
\n
Customizing Risk Score Activity Reports
\n
When you create a Risk Score Activity report, you can apply filters to the report to view specific data. To learn about the filters you can apply, see the screenshot and list below:
\n
 
\n
    \n
  1. \nRisk Score Activity: Use this drop-down menu to select your Risk Score report type. For more information about Risk Score Reports, see our SmartRisk Agent\u2122 and Risk Score Guide.
    2. \n
  2. \nBehavior: Use this filter to select the behavior types to display.
    3. \n
  3. \nRisk Factors: Use this filter to select the risk factors to display.
    4. \n
  4. \nSecurity Types: Use this filter to select the security types to display.
    5. \n
  5. \nAdd Filter: You can customize your Risk Score Activity report with the following additional filters.\n
      \n
    • \nUser Groups: Use this filter to select the user groups to display.
      • \n
    • \nExclude User Groups: Use this filter to select the user groups that you would like to exclude from the report. If a user is included in both the User Groups filter and the Exclude User Groups filter, the user will be excluded from the report.
      • \n
    • \nSearch User: Use this filter to search for a specific user\u2019s data to display.
      • \n
    \n
    6. \n
  6. \nData Source: Use this filter to select the data sources for display.
    7. \n
  7. \nTimeframe: Use this filter to select the date range to display. This date range is based on when the events or behaviors occurred.
    8. \n
\n
After you select your filters, click the Apply Changes button.
\n
\n
\nTip:After you click Apply Changes, you can click Save Report to view the report later. When you click Save Report, your report will automatically display in Reports > Saved Reports. For more information about saving reports, see our Save and Send Reports article.
\n
\n
Viewing Risk Score Activity Reports
\n
After you customize your Risk Score Activity report, your report will display a table that you can further customize to display and sort your results. To learn how to customize the table, see the screenshot and list below:
\n
 
\n
    \n
  1. \nManage Columns: Click this text to display a drop-down menu that contains a list of available columns. From the drop-down menu, you can deselect the default columns and select additional columns to display in the table.\n
     
    \n
    \n
    \nNote:If you export a CSV file of your report, the columns that you\u2019ve selected from the drop-down menu will display in the CSV file. If you save your report by clicking Save Report, the columns you\u2019ve selected will display in the saved report.
    \n
    \n
    2. \n
  2. \nArrow: Click this icon in the column headers to sort the results in the table.
    3. \n
\n
Exporting Risk Score Activity Reports
\n
If you would like to export your Risk Score Activity report, you can use the Export as button near the top-right corner of the filter options to generate a CSV file.
\n
 
\n
This action will open a drop-down menu where you can choose to Generate CSV for a CSV file version of the report table.
\n
In the top-right corner of your console, a notification bell icon will display when your generated report is ready to be downloaded. Your generated reports can also be downloaded at any time from the Download Center subtab. For more information, see the Download Center Overview article.
