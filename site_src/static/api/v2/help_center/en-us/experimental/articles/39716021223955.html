You may receive an access request from a user trying to view a secure email you sent. This request may be because the email has been forwarded to them, or they have tried to view it using an alternative email address.
\n
You will receive an email notification informing you that access has been requested. The user's details will be in the email allowing you to decide whether or not access should be granted.
\n
To grant access, follow the steps below:
\n
    \n
  1. Log in to the Administration Panel.
    2. \n
  2. \n
    Select Sent Packages on the side menu.
    \n
      
    \n
    3. \n
  3. \n
    Select the With Pending Requests tab.
    \n
      
    \n
    4. \n
  4. Select the recipient name under The following users have requested access to this package.
    5. \n
  5. You can then choose to Approve Request or Deny Request.
    6. \n
  6. Ensure you select Save when approving requests or amending permissions.
    7. \n
