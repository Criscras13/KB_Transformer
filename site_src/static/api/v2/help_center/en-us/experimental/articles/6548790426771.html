You can use user information placeholders to populate information from your users' user profiles. You can populate information from any employee, including managers and admins. For general information about placeholders, see our How to Use Placeholders article.
\n
\n
\nRemember:You should only use user information placeholders that you have imported or added to the user's profile. If the column for a field is empty on the user's profile, the placeholder will display a blank space in the email that the user receives.
\n
\n
For information about each of the user information placeholders, see the table below:

Placeholder NamePlaceholder Drop-Down Menu OptionInformation Populated
[[email]]EmailThis placeholder populates the user's email address.
[[first_name]]First Name\n
This placeholder populates the user's first name.
\n
[[last_name]]Last Name\n
This placeholder populates the user's last name.
\n
[[display_name]]Display Name\n
This placeholder populates the user's first name and last name by default.
\n
If the first and last name fields in the user's profile are blank, this placeholder will populate the user's email address instead.
\n
\n
\nNote:You can set how this placeholder populates in your Account Settings. For more information, see our Updating Your KSAT Account Information Settings article.
\n
\n
[[job_title]]Job TitleThis placeholder populates the user's job title.
[[phone_number]]Phone NumberThis placeholder populates the user's phone number.
\n
[[extension]]
\n		ExtensionThis placeholder populates the user's phone extension.
[[location]]LocationThis placeholder populates the user's location.
[[division]]DivisionThis placeholder populates the user's division.
[[manager_name]]Manager Name\n
This placeholder populates the user's manager's name.
\n
\n
\nNote:This placeholder populates the name that is set in the Manager Name field of the user's profile. If you are creating a training notification to send to managers, use the [[first name]] and [[last name]] placeholders to address the manager by name.
\n
\n
[[manager_email]]Manager EmailThis placeholder populates the email address of the user's manager.
