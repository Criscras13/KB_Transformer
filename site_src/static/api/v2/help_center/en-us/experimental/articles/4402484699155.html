You can disable user provisioning for a single user or a group of users. You will need to disable user provisioning prior to making updates for users using the CSV Import feature.
\n
See the sections below for more information about how to disable user provisioning.
\n\n
Disabling User Provisioning for a Single User
\n
Follow the steps below to disable user provisioning for a single user.
\n
    \n
  1. Log in to your KSAT console.
    2. \n
  2. Navigate to the Users tab.
    3. \n
  3. Select the user you wish to remove from user provisioning.
    4. \n
  4. Navigate to their User Information tab. \n\n
    5. \n
  5. In the green banner, select the text that reads click here. You can add a user back to user provisioning by clicking the same text again in the yellow banner. \n\n
    6. \n
\n\n
Disabling User Provisioning for a Group of Users
\n
To disable user provisioning for a group of users, please make sure you have a CSV file that contains all of the users you wish to edit and a column titled Provisioning Managed with True and False values. Follow the steps below to disable user provisioning for a group of users.
\n
    \n
  1. Open your CSV file containing the group of users who need their user provisioning disabled.
    2. \n
  2. Set the Provisioning Managed value to FALSE inside of the CSV file.
    3. \n
  3. Log in to your KSAT console.
    4. \n
  4. Navigate to the Users tab.
    5. \n
  5. Select Import Users.
    6. \n
  6. Use the CSV Import feature to import a CSV file.
    7. \n
  7. Upload your modified CSV file and click Import Users.\u00a0
    8. \n
