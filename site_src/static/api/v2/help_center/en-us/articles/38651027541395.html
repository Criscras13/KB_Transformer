You can integrate your Tidal Cyber platform into your KSAT console. Integrating Tidal Cyber into your KSAT console allows organizations to efficiently assess, organize, and optimize their cyber defenses based on a deep understanding of the threats and adversaries that are most relevant to them. Phishing Security Test (PST) activity can also be integrated into your Tidal Cyber platform.
\n
Before You Connect Tidal Cyber to Your KSAT Console
\n
Before integrating Tidal Cyber into your KSAT console, you must find your API address and create a Reporting Event API key. Follow the steps below to do both actions.
\n
    \n
  1. Your API address depends on where your KnowBe4 account is located. Please see the table below to find the API address for your region.\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n
    RegionBase URL
    US\n
    https://us.api.knowbe4.com
    \n
    CA\n
    https://ca.api.knowbe4.com
    \n
    EU\n
    https://eu.api.knowbe4.com
    \n
    UK\n
    https://uk.api.knowbe4.com
    \n
    DE\n
    https://de.api.knowbe4.com
    \n
    \n
    2. \n
  2. Log in to your KSAT console as an admin.
    3. \n
  3. Navigate to Account Settings in the top-right corner under your account\u2019s email address.
    4. \n
  4. In Account Settings, navigate to Account Integrations.
    5. \n
  5. Under Account Integrations, navigate to API > Reporting API. \n
    6. \n
  6. On the Reporting API tab, click the + Create New API Token button to open the API token creation screen. \n
    7. \n
  7. Enter a name for the token, then click Create Token.
    8. \n
  8. Please save this API token for the below steps.
    9. \n
\n
Available Connections to Tidal Cyber
\n
You can connect KnowBe4 Enterprise Security Training and KnowBe4 Simulated Phishing, also known as Phishing Security Tests, to Tidal Cyber. You can use the same API token to make both connections.\u00a0
\n
\n
Connecting Security Awareness Training to Tidal Cyber
\n
The following KnowBe4 API Endpoint is used to enable this integration:
\n
    \n
  • GET /v1/training/campaigns
    • \n
\n
You can connect your Tidal Cyber platform to your KSAT console by following the steps below.
\n
\n
\nNote:Only Tidal Cyber users with the admin role can enable and configure integrations.
\n
\n
    \n
  1. Navigate to Administration > Integration in the menu on the right side of the screen. Your current Active Connections, including KnowBe4, will appear in a table. \n
    2. \n
  2. Select the Available Integrations tab, select the KnowBe4 Enterprise Security Awareness Training product, and click Connect Integration. \n
    3. \n
  3. You should be prompted to Create KnowBe4 Connection. \n
    4. \n
  4. Under Select Integration Type(s), enter the information required for this integration. In this case, enter \u201cGet Configuration\u201d.
    5. \n
  5. Under API Details, add your API token and select your deployment region\u2019s API address, as outlined in the earlier section.
    6. \n
  6. Select Create to save.
    7. \n
  7. The Active Connection table will now contain the KnowBe4 integration in the Enabled state and will run automatically.
    8. \n
\n
To validate that your integration is working as expected in Tidal Cyber, add KnowBe4 to a defensive stack. After the KnowBe4 integration runs, check the defensive stack for capability toggle changes. You can also confirm the integration made changes by checking the exported integration logs.
\n
\n
Integrating Phishing Security Tests (PSTs) into Tidal Cyber
\n
You can integrate KnowBe4 Phishing Security Test (PST) data into your Tidal Cyber platform. This action ensures users' Tidal Confidence Scores reflect their efforts to stay threat-informed.
\n
You can follow the same steps outlined in the Before You Connect Tidal Cyber to Your KSAT Console and Connecting Tidal Cyber to Your KSAT Console sections above, except select the KnowBe4 Simulated Phishing product and use the following KnowBe4 API Endpoint to enable this integration:
\n
    \n
  • GET /v1/phishing/security_tests
    • \n
\n
To validate your PSTs are connected to your Tidal Cyber platform:
\n
    \n
  1. Navigate to the product registry page and find KnowBe4. \n
    2. \n
  2. When you click under the KnowBe4 product, you should see the capabilities listed. The source should be listed as INTEGRATION-KNOWBE4SP. \n
    3. \n
