For information about the internet browsers KCM GRC supports and the display settings that we recommend, see the sections below.
\n\n
Supported Browsers
\n
KCM GRC supports the latest version of the browsers that are listed below:\u00a0
\n
    \n
  • Google Chrome
    • \n
  • Firefox
    • \n
  • Safari
    • \n
  • Microsoft Edge Chromium-based
    • \n
\n
If you're experiencing issues with KCM GRC while using any of these browsers, please contact our support team for help.\u00a0
\n\n
Recommended Display Settings
\n
KCM GRC supports a minimum screen display resolution of 1920 x 1080 pixels, which is also known as full high definition (FHD).
\n
For the best user experience, you will also need to follow the display recommendations that are listed below:\u00a0
\n
    \n
  • Set your internet browser's zoom setting to 100%.
    • \n
  • In your device's display settings, set your resolution scale to 100%.\u00a0
    • \n
\n
If you're experiencing display issues with KCM GRC while following these display recommendations, please contact our support team for help.\u00a0
