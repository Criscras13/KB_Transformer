You have the option to clone any group in your console, but this feature works differently depending on the type of group you would like to clone.
\n
Click on a group type below to learn how to clone it.
\n
\n
\nNote:If you'd like to clone an archived group, start by restoring the group. See the Archiving Groups section of the Users and Groups article for more information.
\n
\n
How to Clone a Console Group
\n
Cloning a standard group created in the console will create a new group containing the same users. Follow the steps below to clone a console group:
\n
    \n
  1. In your KSAT console, go to the Users tab.
    2. \n
  2. Select the\u00a0Groups subtab.\u00a0
    3. \n
  3. From the listed groups, find the group you would like to clone.
    4. \n
  4. Under the Actions column, click the\u00a0drop-down arrow of the desired group and select Clone. \n
    5. \n
  5. In the window that displays, enter a\u00a0New Group Name and click\u00a0Clone Group. \n
    6. \n
\n
How to Clone a Provisioning-Managed Group
\n
Cloning a group that is managed by Active Directory Integration (ADI) or System for Cross-Domain Identity Management (SCIM) will create a new group containing the same users. However, this new group will not be managed by provisioning. Visit our Active Directory Integration (ADI) Configuration Guide or our SCIM Configuration Guide\u00a0for more information.
\n
Follow the steps below to clone a provisioning-managed group:
\n
    \n
  1. In your KSAT console, go to the Users tab.
    2. \n
  2. Select the\u00a0Groups subtab.\u00a0
    3. \n
  3. From the listed groups, find the group you would like to clone.
    4. \n
  4. Under the Actions column, click the\u00a0drop-down arrow of the desired group and select Clone. \n
    5. \n
  5. In the window that displays, enter a\u00a0New Group Name and click\u00a0Clone Group. \n
    6. \n
\n
How to Clone a Smart Group
\n
When cloning a Smart Group, you have the option to either create a new Smart Group or create a standard console group.
\n
If you choose to create a new Smart Group while cloning, the new Smart Group will have the same criteria. You can change or customize these criteria once the new Smart Group has been created. Visit our Smart Groups Overview for more information.
\n
If you choose to clone a Smart Group to create a standard console group, the new console group will contain the same users but will not have Smart Group capabilities. Follow the steps below to clone a Smart Group:
\n
    \n
  1. In your KSAT console, go to the Users tab.
    2. \n
  2. Select the\u00a0Groups subtab.\u00a0
    3. \n
  3. From the listed groups, find the group you would like to clone.
    4. \n
  4. Under the Actions column, click the\u00a0drop-down arrow of the desired group and select Clone. \n
    5. \n
  5. In the window that displays, select Smart Group or\u00a0Console Group.
    6. \n
  6. Then, enter a New Group Name and click\u00a0Clone Group. \n
    7. \n
