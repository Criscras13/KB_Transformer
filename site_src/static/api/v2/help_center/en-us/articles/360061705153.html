A phish link domain is the URL that displays when a user hovers their mouse over a link in a Phishing Security Test (PST). You can manage your phish link domains by navigating to Phishing > Domains in your KSAT console.
\n
On the Domains subtab, you can view details about your phish link domains in the Phish Link Domains table, download a CSV file with information about your phish link domains, and search for specific phish link domains.
\n
\nNote:If you don't have access to this subtab, you can contact our support team for a list of your phish link domains.
\n
For more information about the Domains subtab, see the screenshot and list below:
\n\n
    \n
  1. \nDomain Name: This column displays the phish link domain\u2019s full domain name.
    2. \n
  2. \nRoot Domain: This column displays the phish link domain\u2019s root domain.
    3. \n
  3. \nAdded: This column displays the date when the phish link domain was added to the KSAT console.
    4. \n
  4. \nDomain Type: This column displays the phish link domain\u2019s type, either System or Custom. The System type is used for phish link domains created by KnowBe4, and the Custom type is used for phish link domains that you\u2019ve created.
    5. \n
  5. \nReply-to: This column will display a checkmark next to phish link domains that are also used for reply-to email addresses in phishing templates.
    6. \n
  6. \nHide Domain: You can turn on this toggle to hide the phish link domain from the Phish Link Domains table.
    7. \n
  7. \nShow Hidden Domains: You can turn on this toggle to view phish link domains that you\u2019ve previously hidden. When you turn on this toggle, all hidden phish link domains will display in gray. If you would like to add a hidden phish link domain back to the Phish Link Domains table, you can turn on the Show Domain toggle at the end of the row.
    8. \n
  8. \nDownload CSV: You can click this button to download a CSV file with information about the phish link domains in the Phish Link Domains table.
    9. \n
  9. \nSearch: You can search for a specific phish link domain by entering a part of the domain\u2019s name into the search bar.
    10. \n
  10. \n+ Create New Phish Link Domain: You can click this button to create a custom phish link domain. For more information about creating custom phish link domains, see our How to Create Custom Subdomains article.
    11. \n
\n
\nNote:All phish link domains listed on the Domains subtab are owned by KnowBe4, including custom phish link domains. Phish link domains are only used for simulated phishing tests.
\n\n
New Available Domains
\n
KnowBe4 will occasionally add new system phish link domains to your KSAT console. When these new phish link domains are added, you will see an announcement in your KSAT console release notes. You can also view any updates to our system phish link domains on our change log.
\n
New phish link domains will be hidden automatically. If you would like to use a new phish link domain, we recommend setting up the necessary whitelisting. Then, you can turn on the Show Hidden Domains toggle so that the domain can be used in your campaign.
