On the Real-Time Coaching subtab of SecurityCoach, you can create and manage real-time coaching campaigns. Real-time coaching campaigns allow you to coach your users about risky activity in real time. When risky activity is detected, your users will receive a SecurityTip with information about the activity and how to avoid it in the future.
\n
For general information about SecurityCoach, see our SecurityCoach Product Manual.
\n
\n
\nTip:To test out your real-time coaching campaign, see our Working with Test Mode Campaigns article.
\n
\n
Before you create your first real-time coaching campaign, we recommend setting up SecurityCoach by following the steps in our Quickstart Guide.
\n
Creating a Real-Time Coaching Campaign
\n
We offer several different methods for creating real-time coaching campaigns, depending on your organization\u2019s needs. You can create a brand new campaign, a campaign based on our recommendations, a campaign based on specific SecurityTip content, or a limited campaign for a specific group of users.
\n
To learn how to create a campaign using each of these methods, see the subsections below.
\n
Creating New Real-Time Coaching Campaigns
\n
To create a new real-time coaching campaign, follow the steps below:
\n
    \n
  1. Log in to your KSAT console.
    2. \n
  2. Navigate to SecurityCoach > Real-Time Coaching.
    3. \n
  3. Click + Create Real-Time Coaching Campaign at the top-right corner of the page.
    4. \n
  4. Fill out the fields on the Create New Real-Time Coaching Campaign page. For more information about these fields, see the screenshot below:\n
    \n
      \n
    1. \nCampaign Name: Enter a name for your real-time coaching campaign.
      2. \n
    2. \nTest Mode: Enable to test how the campaign will perform without sending SecurityTips to your users. You can turn off test mode at any time to begin real-time coaching. For more information, see our Working with Test Mode Campaigns article.
      3. \n
    3. \nDetection Rules: Select one or more detection rules for this campaign using the drop-down menu. For more information, see our Detection Rules Guide.
      4. \n
    4. \nAdditional Criteria: Use this optional drop-down menu to create additional criteria for the campaign. To create criteria, follow the steps below:\n
        \n
      1. Select the Select Criteria drop-down menu.
        2. \n
      2. Select one of the criteria from the drop-down menu.
        3. \n
      3. Complete the fields for the criterion.
        4. \n
      4. Select Save.
        5. \n
      \n
      5. \n
    5. \nSecurityTip Text: Select the SecurityTip notification text template that you would like to send to users enrolled in this campaign. Select the eye iconto preview the notification text template.
      6. \n
    6. \nSecurityTip Content: Select the SecurityTip content that you would like to send to users enrolled in this campaign. Select the image icon to send the SecurityTip as an image, select the GIF icon to send the SecurityTip as a GIF, or select the video icon to send the SecurityTip as a video. You can also select the eye icon to preview the content.\n
      \n
      \nNote: If a SecurityTip is not available as a GIF in the user's default training language, the SecurityTip will be sent as an image. If a localized SecurityTip is unavailable, it will display in English.
      \n
      \n
      7. \n
    7. \nUser Feedback Interactions: Use this section to choose which optional feedback you would like your users to submit for your review. You can enable any of these options:\n
        \n
      • \nConfirmation of Receipt: Allow your users to confirm they read the SecurityTip.\n
        \n
        \nTip: We recommend enabling Confirmation of Receipt for your campaigns to verify that your users have received and read their SecurityTips.
        \n
        \n
        • \n
      • \nQuick Feedback: Allow your users to confirm if they know why they received the SecurityTip.
        • \n
      • \nCustom Feedback: Allow your users to submit custom feedback or questions about the SecurityTip.
        • \n
      \n
      8. \n
    8. \nSecurityTip Delivery Method: Select this drop-down menu to choose how you would like to send the SecurityTip to users. You can select a real-time delivery method and enable email as a backup delivery method. Backup notifications will only be sent if your real-time delivery method cannot be used or is disabled.\n
      \n
      \nNote: The backup delivery method will only be used if your real-time delivery method cannot be used or if it is disabled.
      \n
      \n
        \n
      1. \nSecurityTip Delivery Limit: Select how often you want SecurityTips to be sent to your users.\n
          \n
        • \nLimit to one SecurityTip every 20 minutes: Users won\u2019t receive more than one tip every 20 minutes for this campaign.
          • \n
        • \nLimit to one SecurityTip per day: Users won\u2019t receive more than one tip every 24 hours for this campaign.
          • \n
        \n
        2. \n
      \n
      9. \n
    9. \nSecurityTip Preview: Use this section to preview your SecurityTip content selection, notification template selections, and User Feedback Interactions selections.
      10. \n
    10. \nSend Me a Test: Select this option to send a test SecurityTip using your selected SecurityTip Delivery Method.\n
      \n
      \nNote: The email address you provide for this test must be integrated with your selected SecurityTip Delivery Method. For more information, see our Configuring Delivery Methods articles. If your selected SecurityTip Delivery Method or backup delivery method is email, the email address you provide for this test must be an active user in your KSAT console.
      \n
      \n
      11. \n
    11. \nCreate Campaign: Select Create Campaign to save your settings and create your real-time coaching campaign.
      12. \n
    12. \nCancel: Select Cancel to cancel campaign creation.

Creating Recommended Real-Time Coaching Campaigns
\n
If you\u2019re not sure where to start, SecurityCoach offers recommended real-time coaching campaigns based on your detection rules. These campaigns can be found under the Real-Time Coaching Recommendations section of the Real-Time Coaching subtab.
\n
The Top Recommendation card offers recommendations based on detection rules that are triggered but haven\u2019t been used yet in a campaign. The card also displays the number of times an unused rule has been detected in the past 30 days.
\n
\n
To create a recommended real-time coaching campaign, select the Create Campaign button on a recommended campaign card. You will be taken to the Create New Real-Time Coaching Campaign page. The campaign criteria will already be selected for you, and you will only need to fill out the remaining fields to create the campaign.
\n
Creating Real-Time Coaching Campaigns for SecurityTips
\n
You can also create real-time coaching campaigns for individual SecurityTip content, including any custom graphics that you've uploaded. All of our available SecurityTip content is located in the SecurityCoach tab > SecurityTips subtab.
\n
\n
Some SecurityTip content is available as images, GIFs, or videos. If the GIF format is available, a GIF iconwill display on the SecurityTip card. If the video format is available, a video iconwill display on the SecurityTip card.
\n
To create a campaign for a SecurityTip, select Create Campaign on a SecurityTip card, and you\u2019ll be taken to the Create New Real-Time Coaching Campaign page. The SecurityTip content will already be selected for you, and you will only need to fill out the remaining fields to create the campaign.
\n
Creating Limited Real-Time Coaching Campaigns
\n
If you're not ready to coach your entire organization, you can run a campaign for a specific group of users. For example, you may want to run real-time coaching campaigns for your IT team before the rest of your organization.
\n
To create a limited campaign, follow the steps below:
\n
    \n
  1. While creating a real-time coaching campaign, select User Field in the Additional Criteria drop-down menu. A pop-up window will open.
    2. \n
  2. Edit the fields listed below.\n
      \n
    1. \nUser Field: Select Group Name.
      2. \n
    2. \nCondition: Select Must.
      3. \n
    3. \nComparison: Select Equal.
      4. \n
    4. \nValues: Select the groups you want to include in your test real-time coaching campaign.
      5. \n
    \n
    \n
    3. \n
  3. Select Save.
    4. \n
  4. Finish creating the real-time coaching campaign.
    5. \n
\n
If you don\u2019t want to use groups, you can also limit your campaign recipients by other criteria, such as name or email.
\n
Managing and Editing Real-Time Coaching Campaigns
\n
Once you have created a real-time coaching campaign or a test mode campaign, you can manage or edit the campaign at any time. To manage and edit your real-time coaching campaigns and test mode campaigns, navigate to SecurityCoach > Real-Time Coaching. You can find your existing real-time coaching campaigns under the Real-Time Coaching Campaigns section of the page.
\n
To learn more about the options available in this section, see below:
\n
\n
    \n
  1. \nReal-Time Coaching Campaigns: This table lists your real-time coaching campaigns and test mode campaigns by name. For each campaign, you can see which detection rules are selected, the related vendor for each rule, and the campaign\u2019s status.
    2. \n
  2. \nPreview: Select Preview to preview the SecurityCoach graphic and SecurityTip that users will receive from this campaign.
    3. \n
  3. \nEdit: Select Edit to open the Edit Real-Time Coaching Campaign page. On this page, you can edit your campaign as needed. Then, click the Save button at the bottom-left corner of the page to save your changes.
    4. \n
  4. \nMore Options: Select More Options to view the additional actions for a campaign. These actions include:\n
      \n
    • \nDeactivate: Select this toggle to activate or archive your campaign.
      • \n
    • \nActivate Real-Time Coaching: Select this option to activate a test mode campaign.
      • \n
    • \nDelete: Select Delete to permanently delete a real-time coaching campaign and all associated data. This action is permanent and cannot be undone.

Real-Time Coaching with Phishing Campaigns
\n
SecurityCoach supports integration with your KnowBe4 Security Awareness Training phishing campaigns so that you can send real-time coaching SecurityTips to your users after a phishing failure. For a list of these KnowBe4 system detection rules, see the KnowBe4 Security Awareness Training (KSAT) section of our System Detection Rules by Vendor article.
\n
SecurityCoach also offers a Social Engineering Indicators (SEI) coaching integration with your phishing campaigns. If your users fail a phishing test, SecurityCoach can send them a SecurityTip GIF revealing the Social Engineering Indicators, or red flags, that your users missed from the exact phishing test they failed. Once enabled, these notifications will be sent from a system real-time coaching campaign named Social Engineering Indicators (SEI). For more information, see our Real-Time Coaching with Social Engineering Indicators (SEI) article, SecurityTips Guide, and Social Engineering Indicators (SEI) Guide.
