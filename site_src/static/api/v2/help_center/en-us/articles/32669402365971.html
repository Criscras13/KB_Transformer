In this article, you\u2019ll learn how to integrate your Onyxia platform with your PhishER console. This integration allows you to send information from PhishER to Onyxia to get strategic cybersecurity program recommendations.
\n
\n
\nNote: Before you set up this integration, you'll need to copy the PhishER Product API Token from your KSAT Account Settings. For more information, see the API section of our KnowBe4 Console Account Settings: Account Integrations article.
\n
\n
    \n
  1. Log in to your Onyxia platform.
    2. \n
  2. Navigate to Settings > Integrations.
    3. \n
  3. Search for the KnowBe4 PhishER tile.
    4. \n
  4. Select the tile. You\u2019ll be redirected to the integration setup page.
    5. \n
  5. In the Name field, enter a name for the integration.
    6. \n
  6. In the Description field, enter a brief description for the integration.
    7. \n
  7. In the Data Region drop-down menu, select the base URL based on your region:\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n
    RegionBase URL
    US\n
    https://us.api.knowbe4.com
    \n
    CAhttps://ca.api.knowbe4.com
    EUhttps://eu.api.knowbe4.com
    UKhttps://uk.api.knowbe4.com
    DEhttps://de.api.knowbe4.com
    \n
    8. \n
  8. In the KMSAT Token field, paste the Reporting API key that you copied earlier.
    9. \n
  9. After you\u2019ve filled out all the fields, click Test The Source to make sure the API key is working correctly.
    10. \n
  10. Click Save & Exit to save the integration.
