Below is a list of our new releases and updates to our Active Directory Integration (ADI) feature. If your organization is using an older version of ADI, we recommend that you upgrade to the latest version. The latest version is at the top of this change log.
\n
To upgrade, follow the steps below:
\n
    \n
  1. Download and run the installer from your KnowBe4 Account Settings.
    2. \n
  2. If asked, click Yes to use the data from the previous installation.
    3. \n
\n
The KnowBe4 ADI Sync Service will start automatically once the installation is completed. You can verify that the service is running as expected in the Windows services menu.
\n

Release DateDescription
June 2025\n
Active Directory Integration (ADI) Version 1.10.8.147 was released:
\n
    \n
  • We've updated the Active Directory configuration file to account for special characters in LDAP (group names). We also improved pagination when requesting Active Directory data. This current version of ADI also now supports Windows Server 2025.
    • \n

Release DateDescription
December 2024\n
Active Directory Integration (ADI) Version 1.10.4 was released:
\n
    \n
  • We've updated the Active Directory configuration file so that if any Organizational Units (OUs) are missing, ADI can process them more efficiently. We also addressed the HTTPS and LDAP query timeouts with larger accounts by updating the syntax of the quotation marks so the syncs could run more successfully.
    • \n
\n
May 2024\n
Active Directory Integration (ADI) Version 1.10.0 was released:
\n
    \n
  • We've refactored our code for our ADI tool. The new .exe installation file now features automatic updates, as well as some minor changes to file names and locations. For the most recent information regarding installation, see the Installation and Configuration section of our Active Directory Integration (ADI) Configuration Guide.
    • \n
