The easiest and most efficient way to integrate users from your organization\u2019s Active Directory to your KnowBe4 console is to use our Active Directory Integration (ADI) feature. For more information on this feature, see our Active Directory Integration (ADI) Configuration Guide.
\n
If you prefer to manually integrate your Active Directory information, you can use Dsquery to export your user data as a TXT file. Follow the steps listed below to generate your user list.
\n
    \n
  1. Click Start, then right-click on Command Prompt. From the menu that displays, select Run as administrator.\n
      \n
    • For more information, see this Dsquery article from Microsoft Docs.
      • \n
    \n
    2. \n
  2. Next, run the following command to generate a list of all actual users, excluding disabled accounts. This list will include each user\u2019s first name, last name, and email address.dsquery * -filter \"(&(objectClass=user)(mail=*))\" -attr mail cn -limit 0 > users.txt\n
    3. \n
\n
Once the user list is created, review the data for accuracy. Then, import your users by following the instructions in our Users and Groups article.
