To disable but not remove the Microsoft Outlook add-in, follow the steps below:
\n
    \n
  1. Open Microsoft Outlook.
    2. \n
  2. Navigate to File > Options > Add-ins.
    3. \n
  3. At the bottom of the window under Manage, ensure the COM Add-ins option is selected and select Go.
    4. \n
  4. Unselect the box next to Egress and select OK.
    5. \n
  5. Close down Microsoft Outlook. The add-in should now be disabled.
    6. \n
\n
To remove the Microsoft Outlook add-in, follow the steps below:
\n
    \n
  1. Ensure Microsoft Outlook is closed.
    2. \n
  2. On your device, navigate to Add or Remove Programs.
    3. \n
  3. Under Apps and Features, scroll down to Egress.
    4. \n
  4. Select Egress and then Uninstall.
    5. \n
  5. Follow the steps from the uninstaller.
    6. \n
