In this article, you can learn how to integrate Cortex's XSOAR platform with the PhishER platform. Once you set up this integration, you'll be able to send your PhishER events to XSOAR and do mutations. For example, you can use commands to send PhishER messages and update the message status. Then, you can use this data for your automation and playbooks in XSOAR.
\n
To learn about the benefits of this integration, see our Cortex XSOAR and KnowBe4 document.
\n\n
Configuring the Integration
\n
To configure the integration, follow the steps below:
\n
    \n
  1. Log in to your XSOAR account.
    2. \n
  2. Navigate to Settings > Integrations > Servers & Services.
    3. \n
  3. Search for PhishER.
    4. \n
  4. Click Add instance to create and configure a new integration instance.
    5. \n
  5. Fill out the parameters with information about your PhishER platform. For more information, see the table below:\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n
    ParameterDescriptionRequired
    Your server URL\u00a0True
    API Key\u00a0True
    First Fetch TimeFirst fetch timestamp (&lt;number&gt; &lt;time unit&gt;, e.g., 12 hours, 7 days, 3 months, 1 year)False
    Fetch Incidents\u00a0False
    Fetch LimitMaximum number of alerts per fetch. Default is 50, maximum is 100.False
    Incident type\u00a0False
    Trust any certificate (not secure)\u00a0False
    Use system proxy settings\u00a0False
    \n
    6. \n
  6. Click Test to validate the URLs, tokens, and connection.\u00a0
    7. \n
\n\n\n
Using Commands
\n
After adding the integration, you can use commands to send data from PhishER to XSOAR.
\n
For a list of commands you can use, see the table below:
\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n
 Command  Description 
phisher-message-listUse this command to get messages from PhishER.
phisher-create-commentUse this command to add a comment to a PhishER message.
phisher-update-messageUse this command to update a PhishER message status. The user must provide at least one argument.
phisher-tags-createUse this command to add tags to a given message.
phisher-tags-deleteUse this command to remove tags from a given message.
\n
For more information, see Cortex XSOAR\u2019s KnowBe4 PhishER documentation.
