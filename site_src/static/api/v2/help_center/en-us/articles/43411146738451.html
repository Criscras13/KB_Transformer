As an admin for your business account, you can easily reset user passwords, helping your users regain access to their accounts quickly.
\n
To reset a user\u2019s password, follow the steps below:
\n
    \n
  1. Use admin credentials to log in to your business account here.
    2. \n
  2. Navigate to User Management.
    3. \n
  3. Select the user account that you want to update the password for.
    4. \n
  4. Select the Authentication tab.
    5. \n
  5. From here, you have two options to complete the reset process:\n
      \n
    1. \nAllow the user to reset their own password: Check the Reset user security questions to a random reset code and send a password reset email box. Select Save to generate the reset code and share the code with the user to help them complete the reset process.
      2. \n
    2. \nSet a new password for the user: Select Generate random password. Select whether you want to force the user to change their password when they log in with the generated password or allow the user to reset their password using self-help. Take note of the generated password to share with the user, and select Save.
