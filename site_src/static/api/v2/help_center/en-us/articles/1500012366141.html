Using security roles, you can allow any user to access the ModStore, browse for content, and add content to your account\u2019s library.\u00a0
\n
\nNote: These instructions are for current KnowBe4 customers and require access to our Security Roles feature. If you are not yet a customer, or if you don't have access to our Security Roles feature, please reach out to the KnowBe4 Sales Team.
\n
Follow the steps below to create a ModStore security role:
\n
    \n
  1. First, you must create a group that only includes the users who should have access to the ModStore.\u00a0\n
      \n
    • For detailed instructions, see our How Do I Create a New Group? Article.
      • \n
    \n
    2. \n
  2. Once you have a group created, navigate to the Users tab.
    3. \n
  3. Click on the Security Roles subtab.\u00a0
    4. \n
  4. In the top-right corner of the page, click the +New Security Role button.
    5. \n
  5. Enter a descriptive name for the security role, such as \u201cModStore Access\u201d.
    6. \n
  6. From the Groups drop-down menu, select the group that you created in step 1.\n
    \n
    7. \n
  7. Click on the ModStore tab to edit the ModStore permissions for this security role.\n
      \n
    • For ModStore, select Read/Write.
      • \n
    • For Library, select Read.
      • \n
    • Keep the rest of the ModStore permissions set to No Access.\n
      \n
      • \n
    \n
    8. \n
  8. When you are satisfied with your settings, click the Create Security Role button.
    9. \n
\n
Once this security role is created, the selected users can log in to the KnowBe4 console and access the ModStore tab, add content to your account\u2019s library, and view content that has been added to the library.\u00a0
