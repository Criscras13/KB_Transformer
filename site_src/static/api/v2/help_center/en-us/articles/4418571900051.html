KnowBe4\u2019s integration with Vanta combines KnowBe4\u2019s training results with the Vanta platform, which automatically updates users\u2019 security checklists. You can use this integration to track your KnowBe4 training results, including specific compliance-related training. Once the integration is complete, your Vanta Onboarding settings will automatically recognize KnowBe4 as your default security awareness training.
\n
\nNote: Before you set up this integration, you'll need to copy the Reporting API Token from your KSAT Account Settings. For more information, see the API section of our KSAT Account Settings: Account Integrations article.
\n
To integrate your KnowBe4 training information with Vanta, follow the steps below:
\n
    \n
      \n
    1. Log in to your Vanta admin console.
      2. \n
    2. Navigate to the Connect API Token modal.
      3. \n
    3. Add the Reporting API Token found in your KnowBe4 Account Settings.\u00a0
      4. \n
    \n
\n
\n
    \n
  1. Select which training campaigns you want to track. You can select both active and inactive campaigns.
    2. \n
\n
\n
    \n
  1. Connect each training campaign with a Vanta SAT type by selecting the type of training from the drop-down menu and selecting the desired check box.
    2. \n
\n
\n
    \n
  1. Click Save.
    2. \n
\n
If you're an admin, you can make changes to the integration settings from your Vanta account. Navigate to the Connections page, then click the ... icon on the KnowBe4 tab. You can also track user training progress from the People page.
