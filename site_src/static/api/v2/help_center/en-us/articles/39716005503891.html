Installing Protect for Windows PC will create an add-in within Microsoft Outlook, allowing your users to receive the benefits of the Prevent and/or Protect functionality that your organization has purchased.
\n
The Outlook add-ins for Windows PC can be downloaded here. You can have the choice to deploy the COM add-in or the Web add-in. To decide which add-in meets your organization's requirements, see the Outlook Add-Ins: COM vs Web article.
\n
\n
\nNote:This article covers the installation of the COM add-in. If you choose to deploy the Web add-in, please refer to the Web Add-In-Deployment Guide.Once you have downloaded the add-in, you will need to run the installation file.
\n
\n
    \n
  1. Agree to the terms and conditions and then click Install.
    2. \n
  2. In the next window, click Next.
    3. \n
  3. A progress bar will be displayed while Protect is installing. Once finished, you will see a window that lets you know the installation is complete.
    4. \n
\n
Protect has now been installed and will be visible within Outlook.
