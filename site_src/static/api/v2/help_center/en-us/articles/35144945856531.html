You can integrate your Trustero platform into your KSAT console. Trustero is an AI-driven Governance, Risk, and Compliance (GRC) platform that allows you to better collect evidence for more robust reporting. It has a pre-built integration with KnowBe4\u2019s security awareness training.
\n
Before You Connect Trustero to Your KSAT Console
\n
Before you integrate Trustero into your KSAT console, follow the steps below.
\n
    \n
  1. Log in to your KnowBe4 KSAT console as an administrator.
    2. \n
  2. Go to Account Settings.
    3. \n
  3. Navigate to the Account Integrations option on the sidebar.
    4. \n
  4. Click the API option.
    5. \n
  5. Select the Enable Reporting API Access toggle.
    6. \n
  6. Click the Reporting API link. \n
    7. \n
  7. Click the Create New API Token button.
    8. \n
  8. The console will create an API key. Please save this API key for the next steps.
    9. \n
\n
Connecting Trustero to Your KSAT Console
\n
    \n
  1. Log in to the Trustero portal and navigate to Receptors, Trustero\u2019s evidence collection integrations.
    2. \n
  2. Select Add Receptor.
    3. \n
  3. Choose the KnowBe4 receptor located at the bottom-right corner of the receptor list. \n
    4. \n
  4. You will be brought to the Activate KnowBe4 Receptor page in the Trustero platform. On this page:\u00a0
    \n
      \n
    1. Create a Nickname for the API key that you saved.
      2. \n
    2. Enter the API key saved from Step 8.
      3. \n
    3. Enter your region. You can access your region\u2019s login link in our Learner Support: KnowBe4's Training Instances and Login Links article on our Knowledge Base.
      4. \n
    4. Click Activate. \n
      5. \n
    \n
    5. \n
\n
