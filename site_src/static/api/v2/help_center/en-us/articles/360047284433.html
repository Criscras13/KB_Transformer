The brandable content feature allows you to create a branded theme and apply it to active training campaigns with eligible content. Use the Brandable Content tab to set your brand color, upload a company logo, and add an introduction and final page. These optional pages include your company logo, custom text, and an image of your choice. Use this feature to provide a familiar look and feel for your employees. Read the article below or watch this video to learn more.
\n
Finding Brandable Content
\n
In the ModStore, any content that can be branded has a Brandable Content\u00a0label in the content description.
\n
To find brandable content, follow the steps below:
\n
    \n
  1. Log in to your account and select the ModStore tab.
    2. \n
  2. From the Browse tab, click Additional Filters\n
    3. \n
  3. From the Content Features drop-down menu, select Brandable Content to browse all content that can be branded.
    4. \n
\n
Creating a Branded Theme
\n
To create a brand theme, follow the steps below:
\n
    \n
  1. Log in to your account and navigate to the ModStore tab.
    2. \n
  2. Select the Brandable Content tab.
    3. \n
  3. Click Add Branded Theme.
    4. \n
  4. Customize your branded theme. See Brand Customization for more information on customization options.
    5. \n
  5. Then, click Create.\u00a0
    6. \n
\n
\n
Now, you are ready to apply your theme.
\n
Brand Customization
\n
When creating a new branded theme or editing an existing one, there are three parts you can customize: Theme Settings, Introduction Page, and Final Page.
\n
Theme Settings
\n
When creating a new branded theme, Theme Settings is a required section. You can customize the overall look and feel of the theme by using the\u00a0settings listed below.
\n
    \n
  1. \nTheme Name: A theme name is required. This name will help you identify this theme and is displayed in your list of branded themes.
    2. \n
  2. \nBrand Color: Elements such as the progress bar, active text, and icons will display in this brand color. Enter the HEX code for your brand or click the square icon to the right of this field to open a color selector.
    3. \n
  3. \nCompany Logo: Upload a logo to display at the bottom of the content menu and on the optional introduction and final pages. The image must be a PNG that is 200px wide and 100px tall.\n
      \n
    • For best results, use a logo with a transparent background.
      • \n
    \n
    4. \n
\n
The settings in the screenshot above will result in the following branded theme:
\n
Introduction Page
\n
The Introduction Page is optional and can be customized using the settings listed below.
\n
    \n
  1. \nPage Heading: A heading is required for the introduction page. This field is limited to 100 characters.
    2. \n
  2. \nPage Text: Page text is required for the introduction page. This field is limited to 100 characters.\u00a0
    3. \n
  3. \nPanel Image: Upload an image to display on the left side of the introduction page. The image must be a PNG or JPG that is 512px wide and 768px tall.
    4. \n
\n
The settings in the screenshot above will result in the following branded introduction page:
\n
\n
Final Page
\n
The Final Page is optional and can be customized using the settings listed below.
\n
    \n
  1. \nPage Heading: A heading is required for the final page. This field is limited to 100 characters.
    2. \n
  2. \nPage Text: Page text is required for the final page. This field is limited to 100 characters.\u00a0
    3. \n
  3. \nPanel Image: Upload an image to display on the left side of the final page. The image must be a PNG or JPG that is 512px wide and 768px tall.
    4. \n
  4. \nPhone Number & Email: Provide contact information so that users know who to reach out to with questions.
    5. \n
  5. \nDocument and Policies:\u00a0Select an icon from the Type drop-down menu, then enter your URL. We recommend editing the link description to the name of the document or policy because this text is displayed on the final page.\u00a0\n
      \n
    • You can add up to three links to related documents or policies.
      • \n
    \n
    6. \n
  6. \nSocial Media Links: Select an icon from the Platform drop-down menu. Then, enter your URL. Platform options include Facebook, Instagram, Twitter, YouTube, LinkedIn, Snapchat, Pinterest, and Slack.\n
      \n
    • You can add up to three links to social media pages.
      • \n
    \n
    7. \n
\n

\n
Managing Your Branded Themes
\n
From the ModStore tab, select the Brandable Content tab. Here you will see a table that allows you to view and manage your branded themes. You can also apply these themes, create a new branded theme, and set your default branded theme.
\n
    \n
  1. \nSet Default Theme drop-down menu allows you to set a default branded theme for all campaigns with eligible content. This will apply to any campaign that does not have a different theme selected.
    2. \n
  2. \nBranded Theme Name lists all branded themes. You can click on this or any other header to sort the table to your liking.
    3. \n
  3. \nCampaign Name lists all of the campaigns with the corresponding theme applied.
    4. \n
  4. \nLast Updated is the date on which the theme was last edited.
    5. \n
  5. \nEditopens a page showing the options selected when the theme was created. See Brand Customization for more details.
    6. \n
  6. \nClone allows you to make a copy of the desired theme.\n
      \n
    • The created theme will still need to be applied to campaigns. It will not automatically be applied.
      • \n
    \n
    7. \n
  7. \nDelete will delete the branded theme.\n
      \n
    • Any campaigns that had the deleted theme applied will revert back to the default theme.
      • \n
    • If the default theme is deleted, all campaigns with this theme applied will revert back to the original, unbranded content.
      • \n
    • This action cannot be undone.
      • \n
    \n
    8. \n
\n
Applying a Theme
\n
This\u00a0Apply Themes page lists all active training campaigns and the content within them. To navigate to this page, from the ModStore tab, select the Brandable Content tab. Then, click Apply Theme.
\n
\n
\nNote:To apply a branded theme, the brandable content must be part of an active Training Campaign. See our\u00a0Training Campaigns and Content Management\u00a0article for instructions on setting up a campaign.
\n
\n
\n
    \n
  1. \nHide campaigns without brandable content - Campaigns without Brandable Content are automatically hidden from view. You can choose to show these by clicking the switch at the top of the page.
    2. \n
  2. \nTheme - To apply a different theme than the default to a specific campaign, find the campaign in the list and select the desired theme from the drop-down menu.
    3. \n
  3. \nPreview- Once a theme is applied to a campaign, you can preview the content by clicking Preview and selecting a language.
    4. \n
