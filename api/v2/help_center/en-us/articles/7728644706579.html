If your organization uses KnowBe4's Security Awareness Training (KSAT) console for training, you can integrate your KSAT console with your KCM GRC platform. You can use this integration to create automated KCM GRC tasks that will collect your users' KSAT training completions as evidence. Then, you can use these tasks to show your auditors that your organization is meeting its training requirements.
\n
To set up this integration, you'll need to create a KSAT API token and then configure the integration in your KCM GRC platform. To complete this process, see the sections below.
\n
\n
Creating a KSAT API Token
\n
To connect your KSAT console with your KCM GRC platform, you'll need to create a KSAT API token.\u00a0
\n
To create a KSAT API token, follow the steps below:
\n
    \n
  1. Log in to your KSAT console.
    2. \n
  2. Click your email address at the top-right corner of the page.
    3. \n
  3. From the drop-down menu that opens, select Account Settings.
    4. \n
  4. Navigate to Account Integrations > API.
    5. \n
  5. Click the Product API link. When you click this link, you'll be taken to the Product API subtab. \n
    6. \n
  6. At the top-right corner of the page, click the + Create New API Token button. \n
    7. \n
  7. Fill out the fields on the Create New API Token page.
    8. \n
  8. Click Create Token. The Product API Token pop-up window will open.
    9. \n
  9. Click the Product API token to copy the token. Make sure to save your copied API token somewhere that you can easily access. After you close this window, you will not be able to view this token again.
    10. \n
  10. Click OK.
    11. \n
\n
After you've copied your new KSAT API token, continue to the Configuring the Integration section below.\u00a0
\n
\n
Configuring the Integration
\n
After you copy your API token, you can configure the integration in your KCM GRC platform.
\n
To configure the integration, follow the steps below:
\n
    \n
  1. Log in to your KCM GRC platform.\u00a0
    2. \n
  2. At the top-right corner of the page, select Settings >\u00a0Account Settings.\u00a0\n
    3. \n
  3. On the View Account page, select the Integrations subtab. \n
    4. \n
  4. In the KnowBe4 KSAT Console section of the page, click the Open\u00a0button. When you click this button, the\u00a0KnowBe4 KSAT Console page will open.\u00a0 \n
    5. \n
  5. In the API Token field, paste the API token you copied in the Creating a KSAT API Token section above.
    6. \n
  6. Click the\u00a0Connect button to connect to your KSAT console. \n
    7. \n
\n
After you configure the integration, you can begin creating tasks to collect KSAT evidence. For more information about creating tasks for KSAT, see our KSAT Integration: How to Work with Tasks article.
