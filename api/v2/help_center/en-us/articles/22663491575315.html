The PhishER Blocklist feature integrates with your Microsoft 365 mail server to help you to prevent malicious or spam emails from reaching your users\u2019 inboxes. After you enable the PhishER Blocklist, you will need to assign the Exchange Administrator role to the PhishER Blocklist application to authenticate it. You can assign the role in your Microsoft Entra or Microsoft Azure console.
\n
\n
\nNote:If you enable the Global Blocklist, you will need to assign the Exchange Administrator role to the Global Blocklist application to authenticate it.
\n
\n
Assigning the Role in Microsoft Entra
\n
To assign the Exchange Administrator role to the application in Microsoft Entra, follow the steps below:
\n
    \n
  1. Log in to the Microsoft Entra admin center using your admin credentials. You must have at least a Privileged Authentication Administrator role.
    2. \n
  2. Navigate to Microsoft Entra ID > Identity > Show more > Roles & admins > Roles & admins\u00a0to open the Roles and administrators | All roles window.
    3. \n
  3. Click the Exchange Administrator role name to open the Exchange Administrator | Assignments window. \n
    4. \n
  4. Click Add assignments to open the Add assignments window.
    5. \n
  5. Under the Membership subtab\u2019s Select member(s) section, click No member selected to open the Select a member pop-up window.
    6. \n
  6. In the pop-up window that opens, enter \"PhishER Blocklist\" in the search bar to display the PhishER Blocklist application.
    7. \n
  7. Select the check box next to the PhishER Blocklist application and click Select to return to the Add assignments window. The application will display under the Selected member(s) section.\n
    \n
    \nNote:Depending on your version of Microsoft Entra, you may see different settings for steps 5-7. If the Add assignments pop-up window displays a search bar, enter \"PhishER Blocklist\" in the search bar. Select the check box next to the PhishER Blocklist application and click Add to return to the Add assignments window. Skip the remaining steps.
    \n
    \n
    8. \n
  8. Under the Setting subtab\u2019s Assignment type section, select Active.
    9. \n
  9. Under the Maximum allowed assignment duration is permanent section, select the Permanently assigned check box.
    10. \n
  10. In the Enter justification field, enter a reason for your selected duration.
    11. \n
  11. Click Assign.
    12. \n
\n
Assigning the Role in Microsoft Azure
\n
To assign the Exchange Administrator role to the application in Microsoft Azure, follow the steps below:
\n
    \n
  1. Log in to the Microsoft Portal.Azure admin center using your admin credentials. You must have at least a Privileged Authentication Administrator role.
    2. \n
  2. In the Azure Services Dashboard that opens, click Microsoft Entra ID roles and administrators to open the All roles window. If the icon is not displayed, enter \"Microsoft Entra ID roles and administrators\" in the search bar. \n
    3. \n
  3. Click the Exchange Administrator role name to open the Exchange Administrator | Assignments window. \n
    4. \n
  4. Click Add assignments to open the Add assignments window.
    5. \n
  5. Under the Membership subtab\u2019s Select member(s) section, click No member selected to open the Select a member pop-up window.
    6. \n
  6. In the pop-up window that opens, enter \"PhishER Blocklist\" in the search bar to display the PhishER Blocklist application.
    7. \n
  7. Select the check box next to the PhishER Blocklist application and click Select to return to the Add assignments window. The application will display under the Selected member(s) section.\n
    \n
    \nNote:Depending on your version of Microsoft Azure, you may see different settings for steps 5-7. If the Add assignments pop-up window displays a search bar, enter \"PhishER Blocklist\" in the search bar. Select the check box next to the PhishER Blocklist application and click Add to return to the Add assignments window. Skip the remaining steps.
    \n
    \n
    8. \n
  8. Under the Setting subtab\u2019s Assignment type section, select Active.
    9. \n
  9. Under the Maximum allowed assignment duration is permanent section, select the Permanently assigned check box.
    10. \n
  10. In the Enter justification field, enter a reason for your selected duration.
    11. \n
  11. Click Assign.
    12. \n
