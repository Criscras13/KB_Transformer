You can integrate your Harmonic Security platform into your KSAT console, allowing new workflows for security awareness training. The result is more intuitive training based on users\u2019 actions.
\n
Harmonic Security allows you to insert Harmonic alert data into a webhook endpoint, whether that be SIEM, SOAR, or other ingestion tools. For example, you can use Tines\u2019 webhooks to build workflows that allow users to resolve their own security issues in Harmonic Security\u2019s user intervention features.
\n
Before You Integrate Harmonic Security with KSAT
\n
Before you integrate Harmonic Security with your KSAT console, you need to ensure that KSAT is configured correctly and that you\u2019ve created a Harmonic Protect custom event by setting up a Smart Group. This will trigger events in KSAT based on an alert from Harmonic Security\u2019s platform.
\n
You can use KnowBe4's User Event API to configure your custom event by following the steps below:
\n
    \n
  1. Make a POST request using the .txt file below.\u00a0\n
    \n
    \n
    \n
    \n
    {\n\"name\": \"Harmonic Data Protection\",\n\"description\": \"Track a user's involvement in a Data Protection alert. You can\nuse this information to select event-specific training content from the ModStore\nfor enrollment or to influence a user's Personal Risk Score.\"\n}\t
    \n
    \n
    \n
    2. \n
  2. You should receive an ID in the response, which you will need to save.
    3. \n
  3. Set up a Smart Group in your KSAT console and set the following Custom Event criteria. Note that you will need to select the event type you created in the previous step.\u00a0
    4. \n
\n
\n
Connecting Harmonic Security with Your KSAT Console
\n
Once you have created a Harmonic Protect custom event, you can now insert Harmonic Security alert data into a webhook, creating new workflows for KSAT\u2019s security awareness training. To create a workflow, follow the steps below.
\n
    \n
  1. Log in to the Harmonic Security portal, and click on the Integrations tab at the bottom left corner of the page.
    2. \n
  2. Click on Webhooks. \n
    3. \n
  3. Using Tines, select the webhook input, copy the URL from a Tines story into Harmonic Security\u2019s webhook integration, and subscribe to all alerts. \n
    4. \n
  4. Store the alert data, including Username, Email, Alert Timestamp, and the Alert ID, in a database. Harmonic Security recommends using Notion\u2019s database features, but you may use any database you like.
    5. \n
  5. You can query the database to see how many times a user has triggered an alert in a specific timeframe, such as 24 hours.\u00a0
    6. \n
\n
\n
\nNote:If you are unable to find an appropriate template in the Tines Story Library, you can use the cURL command listed below. This cURL command tells Tines how to handle the APIs and data from KnowBe4 and Harmonic Security.
\n
\n
\n
\n
\n
curl -v \\\n-X POST \\\n--location \\\n\"https://<>/events\" \\\n-H 'Content-Type: application/json' \\\n-H 'Authorization: Bearer <>' \\\n-d '{\"event_type\":\"Harmonic Data\nProtection\",\"target_user\":\"<>\",\"external_id\":\"<>\",\"source\":\"Harmonic\nProtect\",\"occurred_date\":\"<><date(tines_threshold_alerting.body.raisedat, \\='\\' %y-%m-%d\\='%Y-%m-%d\\' )=')'>\"}'
\n
\n
\n
How Harmonic Security Works with Your KSAT Console and Slack
\n
By using Harmonic Security to create workflows that trigger alerts in KSAT, you can send messages designed in Slack via its Block Kit Builder with information about security events to OpSec and users. Brief and targeted mandatory training will also be assigned to users after three security alerts.
\n
    \n
  • When a user receives their first security alert, the SecOps team will receive a notification in their Slack channel.
    • \n
  • If the alert is triggered a second time, the user will receive a message from Harmonic\u2019s data protection virtual expert, Maestro, in Slack with more detailed information on what occurred and how to better protect organization data. 
    • \n
  • If an alert is triggered a third time, the user will be assigned mandatory security awareness training. This training is assigned to the Smart Group created for Harmonic Security events.
