On the Executive Reports tab, you can add, resize, and rearrange widgets to create custom executive reports. The widgets allow you to display your organization\u2019s phishing, training, and Risk Score metrics. You can also add supporting titles, subtitles, text, and tables. The reports can then be printed as a PDF file or scheduled to be printed later.
\n
\n
\nNote: Data from your KSAT console can take up to 24 hours to process and display in your executive reports.
\n
\n
System Executive Report Templates
\n
The Executive Reports tab includes three system report templates. For more information about these reports, see below:
\n
    \n
  • \nSecurity Admin Report: This report template includes information such as your organization\u2019s training completion, Risk Score trend, Phish-prone Percentage, and Phishing Security Test (PST) results.
    • \n
  • \nCISO Report: This report template includes information such as your organization\u2019s Risk Score trend, risk score, training completion, training status, PST results, and Phish-prone Percentage.
    • \n
  • \nTeam Manager Report: This report template includes information such as the Risk Score, training completion, PST results, and Phish-prone Percentage for each of your manager\u2019s teams.
    • \n
\n
To view these report templates, click the drop-down menu in the top-left corner of the Executive Reports tab and then select the report you want to view. While you're unable to change the system report templates, you can still make edits and then save your changes under a new report name.

Creating Executive Reports
\n
To create an executive report, you can create a new report or clone an existing custom report. For more information, see the subsections below.
\n
Creating New Reports
\n
To create a new report, follow the steps below:
\n
    \n
  1. Log in to your KSAT account and navigate to Reports > Executive Reports.
    2. \n
  2. In the top-right corner of the page, click Create Report > New Report.\n
    \n
    3. \n
  3. In the Report\u2019s Name field, enter a name for your report.\n
    \n
    4. \n
  4. Add at least one widget to your report. You can add as many widgets as you would like to customize your report. For more information, see the Adding Widgets section below.
    5. \n
  5. (Optional) Set your report settings. For more information, see the Report Settings section below.
    6. \n
  6. Click Save Changes in the top-right corner of the page.\n
    \n
    7. \n
\n
Cloning Reports
\n
To clone an existing custom report, follow the steps below:
\n
    \n
  1. Log in to your KSAT account and navigate to Reports > Executive Reports.
    2. \n
  2. In the top-left corner of the page, select the report that you would like to clone.\n
    \n
    3. \n
  3. In the top-right corner of the page, click Create Report > Clone Report.\n
    \n
    4. \n
  4. Enter a name for your new report.\n
    \n
    5. \n
  5. Click Save Changes in the top-right corner of the page.
    6. \n
  6. (Optional) Edit the report as needed. For more information, see the Customizing Executive Reports section below.
    7. \n
\n
Customizing Executive Reports
\n
You can add, remove, resize, and rearrange widgets to create custom executive reports. For more information, see the subsections below.\u00a0
\n
Report Settings
\n
You can select report settings that will apply to all widgets on your report. To select your report settings, follow the steps below:
\n
    \n
  1. Log in to your KSAT account and navigate to Reports > Executive Reports.
    2. \n
  2. In the top-left corner of the page, select the report that you would like to edit.\n
    \n
    3. \n
  3. Click Edit Report.\n
    \n
    4. \n
  4. For more information about the report settings, see the screenshot and list below:\n
    \n
      \n
    1. \nFont: Use this drop-down menu to select a font to use for the report.
      2. \n
    2. \nFont Size: Use this drop-down menu to select a font size to use for the report.
      3. \n
    3. \nText Color: Use this drop-down menu to select a text color to use for the report.
      4. \n
    4. \nPDF Layout: Use this drop-down menu to select the layout of your report when generating a PDF file. You can select from the two options below:\n
        \n
      • Select Manual to preview the PDF files and manually adjust the widget layout. These PDF files will be in portrait orientation. For more information, see the Previewing Reports section below.
        • \n
      • Select Automatic to generate PDF files with your current widget layout. These PDF files will be in landscape orientation.
        • \n
      \n
      5. \n
    5. \nUser Groups: Use this drop-down menu to select the user groups to include in this report.
      6. \n
    6. \nOrganization Chart: Use this drop-down menu to search by user name or job title. This filter shows user data for all organization levels under the selected users or job titles. For example, you can search for a specific job title, such as Director or VP. You can also select from the two additional settings below:\n
        \n
      • Select Include All Team Members to include the direct and indirect reports of the selected managers in the report.
        • \n
      • Select Include Selected Manager to include the selected manager in the report. This check box will aggregate the data of the selected manager and any selected team member in the report.
        • \n
      \n
      7. \n
    7. \nDate Range: Use this drop-down menu to select a date range for the report.
      8. \n
    \n
    5. \n
  5. Click Save Changes in the top-right corner of the page.\n

Setting a Default Executive Report
\n
You can set a default executive report. By default, this report will display when you access the Executive Reports tab. For example, if you have a report that you view daily and another report that you view monthly, you can set the report that you view daily as the default report to save time.
\n
To set a default report, follow the steps below:
\n
    \n
  1. Log in to your KSAT console and navigate to Reports > Executive Reports.
    2. \n
  2. In the top-left corner of the page, select the report that you would like to set as your default report.\n
    \n
    3. \n
  3. Select Set as Default.\n

Downloading Executive Reports
\n
You can generate PDF files of your executive reports to download and print. By default, you have the option to preview the PDF files and manually adjust the layout of the widgets. But you also have the option to use your current widget layout instead. For more information, see the subsections below.
\n
Previewing Reports
\n
You can preview a report and make changes before generating a PDF file. To preview a report, follow the steps below:
\n
\n
\nNote: The Preview PDF button will only display if the Manual option is selected for the PDF Layout report setting. For more information, see the Report Settings section above.
\n
\n
    \n
  1. Log in to your KSAT console and navigate to Reports > Executive Reports.
    2. \n
  2. Create a new report or edit an existing report.
    3. \n
  3. Click Preview PDF to see how your report will look as a PDF file.\n
    \n
    4. \n
  4. (Optional) Make any needed changes to your report. For example, you can resize widgets to fit on one page instead of being divided across two pages.\n
    \u00a0
    \n
    5. \n
  5. Click Preview PDF to exit the preview.
    6. \n
  6. If you made any changes, click Save Changes.
    7. \n
