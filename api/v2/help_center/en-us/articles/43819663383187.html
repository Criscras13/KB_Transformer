You can integrate your Seemplicity platform into your KSAT console. The Seemplicity platform is a Remediation Operations (RemOps) platform that accelerates risk reduction and streamlines security compliance workflows. It helps organizations manage and automate remediation efforts across multiple teams, improving efficiency, effectiveness, and speed. Integrating with KSAT allows you to receive findings about training completion status via the Seemplicity platform.
\n
Before You Connect Seemplicity to Your KSAT Console
\n
Before integrating Seemplicity into your KSAT console, you must know the region where you are connecting to KSAT and obtain your organization\u2019s API token. You must create a KnowBe4 instance API token and save it in a safe location for use in the following steps. For more information on KnowBe4\u2019s API tokens, see our KnowBe4 API Reference Guide.
\n
Connecting Seemplicity to Your KSAT Console
\n
You can connect your Seemplicity platform to your KSAT console by following the steps below.
\n
    \n
  1. In your Seemplicity platform, navigate to Settings > Integrations > Data Sources > KnowBe4 > Integrate.\n
      
    \n
    2. \n
  2. On the New Instance page, enter a descriptive name for this integration in the Name field.\n
      
    \n
    3. \n
  3. From the Region drop-down menu, select the region where you want to connect. Choices include CA, DE, EU, UK, and US.
    4. \n
  4. In the API Token field, paste the KnowBe4 API token that was previously created.
    5. \n
  5. When finished, select Save.
    6. \n
\n
\n
\nNote:Additional steps may be necessary when setting up this integration in the Seemplicity platform. Please refer to the documentation in your Seemplicity platform for more information.
