Allow and Deny lists can be used to specify what happens when specific email addresses, domains, or IP addresses send emails to your organization.
\n
Configurable Defend Admin Lists
\n
There are three types of lists that can be configured in the Defend console:
\n
    \n
  • Allow list\n
      \n
    • Entries will no longer have suspicious or dangerous banners added to emails, regardless of their classification.
      • \n
    \n
    • \n
  • Deny list\n
      \n
    • Entries will have suspicious or dangerous banners added, regardless of Defend's classification.
      • \n
    \n
    • \n
  • Productivity allow\n
      \n
    • Entries will no longer have the graymail or spam banners added, regardless of Defend's classification or user preference.
      • \n
    \n
    • \n
\n
Configured entries for these lists are displayed with the date of their creation and the option to edit or delete them. Entries can be created on the Allow/Deny List or Recent Emails pages.
\n
\n
\nNote:  If link rewriting is enabled for emails processed by Defend, the URLs will be rewritten regardless of allow list entries.
\n
\n
Create Entries on the Allow/Deny List
\n
To create entries on the Allow/Deny list page, follow the steps below:
\n
    \n
  1. Navigate to Allow / Deny lists.
    2. \n
  2. Select the Allow list, Deny list, or Productivity allow tab.
    3. \n
  3. Select Add item.
    4. \n
  4. Select to create the entry By email address, Domain, or IP Address.
    5. \n
  5. Complete the Rule Name and the Email Address / Fully Qualified Domain / IP Address fields.
    6. \n
  6. For email address and domain entries, use the drop-down menu to select the level of Required Authentication.
    7. \n
  7. Select whether to Hide External Banner if required.
    8. \n
  8. Select Save.
    9. \n
\n
\n
\nNote:  Applying the strictest form of authentication possible to Allow List entries is recommended.
\n
\n
Create an Entry on the Recent Emails Page
\n
To create an entry on the Recent Emails page, follow the steps below:
\n
    \n
  1. Navigate to the Recent Emails page.
    2. \n
  2. Select an email entry. A side panel will open.
    3. \n
  3. Select the Add to deny list or Add to allow list button.
    4. \n
  4. An entry for the selected list will be automatically created for the email sender.
    5. \n
