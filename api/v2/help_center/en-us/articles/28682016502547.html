Student Edition is a ModStore add-on option for current KnowBe4 customers. Student Edition content gives your students the tools they need to help mitigate security risks at their organization.
\n
\n
\nImportant: Additional minimum subscription purchasing requirements may apply.
\n
\n
Creating User Groups for Student Training
\n
For the best results, we recommend creating a user group that only includes students. Make sure to name it something easy to find, such as \u201cStudents Only.\u201d For more detailed instructions, see our Create a Group article. If you have a Platinum or higher subscription, you can also automate this process using Smart Groups. For more information on Smart Groups, see the Smart Groups Overview article.
\n
Creating a Security Role for Students
\n
We recommend creating a specific, dedicated security role for managing students. For more information on security roles, see our Security Roles Guide article.
\n
Follow the steps below to create a Security Role:
\n
    \n
  1. Log in to your KnowBe4 console and navigate to the Users tab.
    2. \n
  2. Click on the Security Roles subtab.
    3. \n
  3. In the top-right corner of the page, click the +New Security Role button.
    4. \n
  4. Enter a descriptive name for the Security Role, such as \u201cStudent Management\u201d
    5. \n
  5. Select the student group you created in the above section from the Groups drop-down menu. \n
    6. \n
  6. Click on the Training subtab and edit the training permissions of this Security Role.\n
      \n
    • Keep all the Training permissions set to No Access. \n
      • \n
    \n
    7. \n
  7. Click on the ModStore subtab to edit the ModStore permissions for this Security Role.\n
      \n
    • Keep all the ModStore permissions to No Access. \n
      • \n
    \n
    8. \n
  8. When you are satisfied with your settings, click the Create Security Role button.
    9. \n
