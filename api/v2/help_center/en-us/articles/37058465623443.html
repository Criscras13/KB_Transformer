Spam is unsolicited bulk email, also known as junk email. It's any kind of unwanted and unsolicited digital communication that gets sent out in bulk. Spam is usually sent for commercial purposes and can be dangerous when it contains malicious links or attachments. Defend can detect and action spam emails with the following functionality:
\n
    \n
  • Spam banners
    • \n
  • Moving mail classified as spam to the Junk folder
    • \n
  • Report as spam or Report as not spam buttons on the email summary page
    • \n
\n
\n
\nNote:The options to move and report spam are turned off by default.
\n
\n
Prerequisites
\n
    \n
  • A Defend user group in Microsoft 365.
    • \n
  • Both the Admin Features and User Analysis options are successfully linked to a tenant on the Defend Settings page.
    • \n
\n
Spam User Group
\n
The Egress_Defend_Spam_Users group contains users who receive spam protection functionality in Defend. This group determines which users benefit from spam filtering features and notifications. Custom group names can be specified while the spam feature is disabled by entering the group's mail address in the Spam User Group field.
\n
  
\n
\n
\nNote:The Defend user group can be a static Microsoft group, a dynamic Microsoft 365 group, or a regular distribution list. Due to Microsoft limitations, you cannot nest other groups in Microsoft 365 groups or distribution lists. Dynamic distribution lists are not supported.
\n
\n
Configuration
\n
Follow the below steps to enable your spam options:
\n
    \n
  1. Ensure the prerequisites have been met.
    2. \n
  2. Log in to the Defend console.
    3. \n
  3. Navigate to Settings > Spam.
    \n
      \n
    1. Enable Move Spam Messages using the dropdown menu. This setting ensures emails classified as spam will be automatically moved to the Junk folder.
      2. \n
    2. Enable the Spam Banner using the drop-down menu. This setting is enabled by default and ensures emails classified as spam will have the appropriate banner displayed.
      3. \n
    3. Enable Allow users to Report Spam/Not Spam using the dropdown menu. This setting allows users to customize their experience using the reporting buttons found on the Email Summary page.\n
        
      \n
      4. \n
    \n
    4. \n
  4. Click Save Changes.
    5. \n
