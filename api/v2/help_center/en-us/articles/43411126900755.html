If you've recently purchased a KnowBe4 product for multiple users or added more licenses to your Business Account, follow this article for a straightforward guide on how to invite your team.
To invite a single user to be part of your business account, follow the steps below:
    \n
  1. Use admin credentials to log in to your business account here.
    2. \n
  2. Navigate to Business Account > Invite Users.
    3. \n
  3. Select Invite Single User.
    4. \n
  4. Complete the fields as below:
      \n
    1. \nFirst name, Last name, and Email address: These are required fields. Enter the email address the user will use to access the KnowBe4 product functionality.
      2. \n
    2. \nDirectory location: If you have an organizational structure, you can specify which Organizational Unit (OU) you want the user to be added to.
      3. \n
    3. \nRole: Select which role you want to assign to the user. For more information, see our Understanding Business Account Roles article.
      4. \n
    4. \nPassword: You can enter a password for the user, or leave the password field blank to allow the user to create their own.
      5. \n
    5. \nNote: Add a personalized note to the user that they will receive with their invitation email.
      6. \n
    \n
    5. \n
  5. Select Invite to send the invitation to the user. All invited users are under the Pending Invitations section on the Invite Users page.
    6. \n
Once the user has followed the link in the invitation email, they will no longer be listed under Pending Invitations. They will now be in the User Management section.
\nNote:If the user doesn't accept when the first email is sent, reminders are automatically sent out once a week. After one month, the invitation expires and will need to be resent.
To invite multiple users to be part of your business account, follow the steps below:
    \n
  1. Use admin credentials to log in to your business account here.
    2. \n
  2. Navigate to Business Account > Invite Multiple Users.
    3. \n
  3. Enter the details of the users you want to invite in the format \u201cfirst name, middle name, last name, email address\u201d. Ensure you separate each value with a comma. The middle name field can be left blank and comma-separated. An example entry with no middle name is: John, \u00a0, Smith, johnsmith@example.com.
    4. \n
  4. Alternatively, select the Browse button to invite multiple users simultaneously by importing a CSV file. The default format for this file is \u201cfirst name, middle name, last name, email address\u201d. 
    5. \n
  5. Complete the following fields: 
      \n
    1. \nDirectory location: If you have an organizational structure, you can specify which Organizational Unit (OU) you want the user to be added to.
      2. \n
    2. \nRole: Select which role you want to assign to the user. For more information, see our Understanding Business Account Roles article.
      3. \n
    3. \nNote: Add a personalized note to the user that they will receive with their invitation email.
      4. \n
    \n
    6. \n
  6. Select Invite to send email invitations to the listed users.
    7. \n
