User Feedback is a SecurityCoach feature that allows you to request and receive feedback from your users about the SecurityTips they receive. You can allow your users to confirm they received the SecurityTip, confirm if they know why they received the SecurityTip, or submit custom feedback and questions about the SecurityTip.
\n
Enabling User Feedback
\n
When you create a new real-time coaching campaign, you have the option to enable User Feedback Interactions for your users to interact with when they receive SecurityTips. For more information, see the screenshot and list below:
\n
  
\n
    \n
  • \nConfirmation of Receipt: Enable this option to allow your users to confirm they read the SecurityTip.
    • \n
  • \nQuick Feedback: Enable this option to allow your users to confirm if they know why they received the SecurityTip.
    • \n
  • \nCustom Feedback: Enable this option to allow your users to submit custom feedback or questions about the SecurityTip.
    • \n
\n
For more information, see our Real-Time Coaching Campaigns Guide.
\n
User Feedback Activity Reports
\n
When one or more types of user feedback are enabled for your real-time coaching campaign, you can view your users\u2019 responses in the campaign\u2019s Feedback Activity subtab. Navigate to SecurityCoach > Real-Time Coaching, select the name of your campaign, then select the Feedback Activity subtab.
\n
  
\n
Here, you can use filters to customize and generate reports for your user feedback. For more information on these reports, see our Real-Time Coaching Campaigns Guide.
\n
User Experience
\n
The SecurityTip Preview section of your real-time coaching campaign allows you to preview what a SecurityTip will look like with your selected user feedback. For more information, see our Real-Time Coaching Campaigns Guide.
\n
To view an example SecurityTip with all user feedback enabled, see the screenshot below:
\n
  
\n
For more information on your users\u2019 experience and learner support for user feedback, see our Learner Support: SecurityCoach Notifications article.
