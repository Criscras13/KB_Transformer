\n
\n
\nNote:The information in this guide is the default configuration of Protect. Your experience may vary depending on your organization\u2019s configuration choices.

\n
Encryption
\n
Protect integrates with your Microsoft Outlook and allows you to control and protect the information and data that you send.
\n
When composing a new email, an Unclassified button is displayed. This allows you to select the appropriate level of email security for the email you compose.
\n
When selecting the Unclassified button, the following encryption options are available by default:
\n
  
\n
Large File Transfer
\n
When composing a new email, an Attach Large Files button is displayed. Selecting Attach Large Files opens a side panel. Here, you can drag and drop or browse and select files to attach.
\n
  
\n
Note:
\n
    \n
  • By default, files greater than 10MB will be attached using LFT (this threshold is customizable and may be greater than 10MB). The side panel opens automatically to display the attachments.
    • \n
  • The recommended maximum file size for large file transfer is 2GB.
