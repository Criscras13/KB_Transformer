The Team Dashboard is a customizable dashboard designed for managers to view in their Learner Experience (LX). The default Team Dashboard displays the manager's team, which includes their direct reports. Each manager can also view the Team Dashboard for up to three levels of managers below them in their organization's management hierarchy.
\n
On the dashboard, managers can view the training progress for the users on the selected team. They can also view the status of the users' optional learning, if Optional Learning is enabled in your organization's account. You have the option to include the team\u2019s phishing results and combined Risk Score too. All the data displayed on the Team Dashboard is from the last 12 months.
\n
\n
\nNote: The Team Dashboard only displays data from active campaigns.
\n
\n
\n
\n
\nNote: If you are looking for information about your Team Dashboard, see our Team Dashboard Overview for Managers article.
\n
\n
Click the links below to learn how to set up the Team Dashboard feature.
\n
Before You Begin
\n
When users are imported into the KSAT console, user-specific information is typically imported as well. This information includes a Manager field that determines which users are managers and which users report to those managers.
\n
Before enabling Team Dashboards, make sure that the users in your console have managers listed in their user information. You can verify your users\u2019 information by reviewing the Manage Users page in your admin console. To review this information, follow the steps below:
\n
    \n
  1. In your KSAT console, navigate to the Users tab.
    2. \n
  2. In the top-right of the Manage Users page, click the gear icon.
    3. \n
  3. From the drop-down menu that opens, select Manager Email.
    4. \n
  4. The Manager Email column will now display in the user list. Review this column to verify that your users have managers listed \n
    5. \n
\n
Alternatively, you can generate a CSV file of all of your users and their information by clicking the green Generate CSV link at the top of the Manage Users page. For more information, see our Users and Groups article.
\n
If your users do not have managers listed, you can update this information by using one of the methods listed below:
\n
    \n
  • If you use a user provisioning integration such as Active Directory or SCIM in your account, you must update your users' manager information directly from your identity provider.
    • \n
  • If you do not use a user provisioning integration in your account, you can create and import a CSV file to add manager information. For more information about this process, see our How to Import Users with a CSV File article.\n
    \n
    \nImportant: When creating your CSV file, only use the Email and Manager Email column headers.
    \n
    \n
    • \n
\n
Enable Team Dashboards
\n
When Team Dashboards are enabled, any user who is listed as another user\u2019s manager will see their Team Dashboard on the Dashboard tab of the LX. To enable and customize your organization\u2019s Team Dashboards, follow the steps below:
\n
    \n
  1. From your KnowBe4 admin account, click on your email address at the top-right of the page and select Account Settings.
    2. \n
  2. In your Account Settings, navigate to Training > Learner Experience > Team Dashboard.
    3. \n
  3. Under Team Dashboard, select the Enable Team Dashboard check box.
    4. \n
  4. Once you have selected the Enable Team Dashboard check box, you can customize the Team Dashboards using any of the optional settings listed below:\n
      \n
    1. \nLimit User Information Shown: By default, the Team Dashboard lists each user with details about their individual contributions to the team\u2019s statistics. To only show aggregated information for the team, turn on the Limit User Information Shown toggle.
      2. \n
    2. \nEnable Phishing Information: To display the team\u2019s phishing results, select the Enable Phishing Information check box.
      3. \n
    3. \nEnable Risk Information: To display the team\u2019s current combined Risk Score, select the Enable Risk Information check box.
      4. \n
    \n\n
    5. \n
  5. Click the Save Changes button at the bottom of the page.
    6. \n
\n
Notify Managers
\n
Once you enable the Team Dashboard, we recommend that you send a message to managers to notify them about their new dashboards.
\n
For example, you can copy and paste the message below to send to managers. Before you send the message, make sure that it applies to your organization. You can also customize the message by adding or removing information.
\n
\n
\nTip:To send a message to managers, you can use the User Messaging feature. For more information, see our User Messaging Guide.
\n
