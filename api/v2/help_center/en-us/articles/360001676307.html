If you\u2019re configuring Active Directory Integration (ADI) for your organization, you may need to view the mail and proxyAddresses fields in your Active Directory (AD). These fields can be used to sync user information from your AD to your KSAT console. If your users aren\u2019t populating correctly in your KSAT console or if you receive error messages related to user information, you may need to update these fields.
\n
In this article, you\u2019ll learn how to view and update the mail and proxyAddresses fields in your AD. For general information about fields in your AD, see our Active Directory Integration (ADI) Advanced Configuration Guide
\n
View the Mail Field
\n
To view or update the mail field in your AD, follow the steps below:
\n
    \n
  1. In your AD, navigate to the Active Directory Users and Computers window.
    2. \n
  2. Click View at the top of the page.
    3. \n
  3. From the drop-down menu that opens, select Advanced Features. \n
    4. \n
  4. Double-click on the user whose mail field you would like to view or update.
    5. \n
  5. In the Properties pop-up window that opens, select Attribute Editor.
    6. \n
  6. To view the user's email address, locate mail under the Attribute column.
    7. \n
  7. Select mail, then click the Edit button at the bottom of the pop-up window. \n
    8. \n
  8. In the String Attribute Editor pop-up window that opens, you can enter a new value for the mail attribute. If you enter a new value, click OK to save your changes. \n
    9. \n
\n

View the ProxyAddresses Field
\n
To view or update the proxyAddresses field in your AD, follow the steps below:
\n
    \n
  1. In your AD, navigate to the Active Directory Users and Computers window.
    2. \n
  2. Click View at the top of the page.
    3. \n
  3. From the drop-down menu that opens, select Advanced Features. \n
    4. \n
  4. Double-click on the user whose proxyAddresses field you would like to view or update.
    5. \n
  5. In the Properties pop-up window that opens, select Attribute Editor.
    6. \n
  6. To view the user's proxy addresses, locate proxyAddresses under the Attribute column.
    7. \n
  7. Select proxyAddresses, then click the Edit button at the bottom of the pop-up window. \n
    8. \n
  8. In the Multi-valued String Editor pop-window that opens, you can add or remove a value to the proxyAddresses attribute. If you add or remove a value, click OK to save your changes. \n
    9. \n
