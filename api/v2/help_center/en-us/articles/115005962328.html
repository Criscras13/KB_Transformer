\nNote:We\u2019ve added advanced features to phishing templates! All accounts will be opted in to phishing template advanced features in 2025. For more information, see our Phishing Templates Advanced Features Guide.
\n
\n
You can create a clone of any phishing or training campaign as a starting point for setting up a new test.
\n
The clone will contain all of the same settings, with the exception of the Start Date and End Date (if applicable) which you will have to set manually.
\n
How to Clone a Phishing Campaign
\n
In order to clone a phishing campaign, follow the steps below:
\n
    \n
  1. From your KSAT account, navigate to Phishing > Campaigns.
    2. \n
  2. Click the Actions drop-down menu to the right of the campaign you'd like to copy.
    3. \n
  3. Click Clone.
    4. \n
  4. The Clone Phishing Campaign page will open, will allow you to set up your cloned campaign.\n
      
    \n
    5. \n

How to Clone a Training Campaign
\n
In order to clone a training campaign, follow the steps below:
\n
    \n
  1. From your KSAT account, navigate to Training > Campaigns.
    2. \n
  2. Click the Actions drop-down menu to the right of the campaign you'd like to copy.
    3. \n
  3. Click Clone.
    4. \n
  4. The Clone Training Campaign page will open, which will allow you to set up your cloned campaign.\n
