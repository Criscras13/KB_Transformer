Data Storage Locations\u00a0
KnowBe4 operates within Amazon Web Services (AWS) and Microsoft Azure. AWS follows the Shared Responsibility Model. AWS is responsible for the security of the cloud, and KnowBe4 is responsible for security in the cloud. Information regarding the compliance of AWS data centers can be found on the AWS compliance website. If you are required to review the data center SOC report, you can review the latest AWS SOC 3 report.\u00a0
Similarly, Azure also follows the Shared Responsibility Model. Full details of the compliance of Azure data centers reside on their Azure compliance documentation webpage. You can also download and review their latest Azure SOC 3 report.\u00a0
You are able to select your data storage location based on your data localization requirements.. However, please note that we use ancillary services for certain functions, so these services may store data in another location. View sub-processors
Data is not shared between the data centers. You may request an account in each region, but these will be independent of each other, and data will not synchronize between accounts.
