In the KCM GRC Vendor Risk Management module, you can create questionnaires to send to the vendors or third parties your organization works with. If the questionnaire templates we offer do not include the questions you would like to ask, you can create custom questions to add to your questionnaires.
\n
You can create custom questions outside of a questionnaire, or you can create custom questions when you create the questionnaire. To create custom questions when you create the questionnaire, you can add the questions individually or you can import the questions with a CSV file.
\n
For general information about custom questions and to learn how to create custom questions outside of a questionnaire, see the Creating Custom Questions section of our How to Use the Custom Questions Tab of Your Platform article.
\n
To learn how to create and configure custom questions in a questionnaire and how to create a custom questionnaire template, see the sections below.
\n\n
Adding Individual Custom Questions
\n
When you create a questionnaire, you can create individual custom questions for the questionnaire.
\n
\n
\nNote: All of the questions that you create in your platform will automatically be added to the Custom Questions tab. For more information, see our How to Use the Custom Questions Tab of Your Platform article.
\n
\n
To create and configure individual custom questions in a questionnaire, follow the steps below:
\n
    \n
  1. Create a questionnaire. For instructions, see the Creating Questionnaires section of our How to Create and Configure Questionnaires article.
    2. \n
  2. In the Add Questions From section of the page, click the plus button. When you click this button, a\u00a0Create Question pop-up window will open.\n\n
    3. \n
  3. Enter your question into the Type your question here... field.\u00a0\n
    \n
    \nTip: You can format your questions on multiple lines by pressing the Return or Enter key on your keyboard.
    \n
    \n
    4. \n
  4. Click the drop-down menu next to the field, then select the type of answer option you would like to use for the question. For information about each type of answer option, see the list below:\n
    \n
    \nTip: You can include questions that are informational and do not need to be assigned points. Select the Informational check box to mark a question as informational.
    \n
    \n
      \n
    1. \nFree Form Text: A blank field will display, and users can enter any answer.
      2. \n
    2. \nMultiple Choice: Multiple options will display with radio buttons next to each option. Users can only select one option.
      3. \n
    3. \nCheckbox: Multiple options will display with check boxes next to each option. Users can select one or more options.
      4. \n
    4. \nYes / No / N/A: Yes,\u00a0No, and\u00a0N/A options will display. Users can only select one option.
      5. \n
    \n\n
    5. \n
  5. Configure and add each question. Depending on the type of answer option you selected in the step above, the process for configuring the question will vary. For more information, see the Configuring Questions by Answer Type section below.\n
    \n
    \nNote: The number of points that you assign to each question must be a positive, whole number.
    \n
    \n
    6. \n
  6. Continue adding questions to the questionnaire by repeating step 2 through step 5 above.
    7. \n
  7. When you are finished adding and configuring the questions, click the Mark as Configured button. When you click this button, your questionnaire will be in the Configured status.\n\n
    8. \n
\n
As a final step before you can send your questionnaire to vendors, the questionnaire will need to be marked as Reviewed. For more information, see the Reviewing and Finalizing Questionnaires section of our How to Create and Configure Questionnaires article.
\n\n
Importing Custom Questions
\n
When you create a questionnaire, you can import custom questions in bulk by uploading a CSV file.\u00a0
\n
\n
\nNote: All questions that you import into a questionnaire will automatically be added to the Custom Questions tab of your platform. For more information, see our How to Use Custom Questions in Questionnaires article.
\n
\n
See the subsections below to learn how to format your CSV file, import the CSV file into a questionnaire, and configure the imported questions.
\n\n
Creating a CSV File
\n
When you create the CSV file that contains your custom questions, make sure the CSV file meets the requirements listed below:
\n
\n
\nNote: If you are using Microsoft Excel to create your CSV file, you must save your CSV file in the CSV UTF-8 format.
\n
\n
    \n
  • The separator must be a comma.
    • \n
  • The CSV file must contain the following header line: question,type\n
    \n
    \nNote: The question and type headers are mandatory. You can also add an is_informational header by entering question,type,is_informational for the header line.
    \n
    \n
    • \n
  • The question fields have a character limit of 1,000 characters.
    • \n
  • The type fields must contain one of the entries listed below and must be in the italic format:\n
    \n
    \nImportant: If the type input fields do not match one of the answer types listed below, the type of answer option that is selected for the question will default to Free Form Text.
    \n
    \n
      \n
    • free
      • \n
    • checkbox
      • \n
    • multi
      • \n
    • boolean
      • \n
    \n
    • \n
  • If you include the is_informational header, the fields under the header must contain either \"0\" for questions that are not informational or \"1\", for questions that are informational.
    • \n
\n\n
Importing the CSV File
\n
After you create your CSV file, you can import the CSV file into a questionnaire.\u00a0
\n
To import the CSV file into a questionnaire, follow the steps below:
\n
    \n
  1. Create a questionnaire. For instructions, see the Creating Questionnaires section of our How to Create and Configure Questionnaires article.
    2. \n
  2. In the Add Questions From section of the page, click the upload button. When you click this button, an\u00a0Import Questions pop-up window will open. \n\n
    3. \n
  3. Click the Click to Upload button.
    4. \n
  4. Select your CSV file from your file manager.
    5. \n
  5. Click the Add button to import the questions into your questionnaire.\n\n
    6. \n
\n
\n
\nTip: To view the questions that you've imported, click the All drop-down menu. Then, select Custom Questions.
\n
\n\n
Configuring Imported Questions
\n
After you import custom questions into your questionnaire, you will need to configure points for each answer. When your vendors complete the questionnaire, the points that you assign to each question will be used to score their answers.
\n
To configure points for answers to imported questions, follow the steps below:
\n
    \n
  1. From the navigation panel, navigate to Vendor Management\u00a0>\u00a0Questionnaires.
    2. \n
  2. From the Name column, select the questionnaire that you would like to configure.
    3. \n
  3. In the Questionnaire Versions section of the page, click the Configure button.\n\n
    4. \n
  4. From the Questions section of the page, click the pencil icon button to edit and configure each question.\u00a0\u00a0\n\n
    5. \n
  5. Configure each question. The process for configuring questions will vary based on the answer type. For more information, see the Configuring Questions by Answer Type section below.\u00a0\n
    \n
    \nTip: If you are setting the same point values for all questions, skip this step and click the Set For All button after all questions have been added.
    \n
    \n
    6. \n
  6. Repeat step 4 and step 5 above until you've configured all of the questions in the questionnaire.
    7. \n
  7. Click the Mark as Configured button at the top-right corner of the page. When you click this button, your questionnaire will be in the Configured status.\n\n
    8. \n
\n
As a final step before you can send your questionnaire to vendors, the questionnaire will need to be marked as Reviewed. For more information, see the Reviewing and Finalizing Questionnaires section of our How to Create and Configure Questionnaires article.
\n\n
Configuring Questions by Answer Type
\n
When you configure questions in your questionnaire, the process will vary based on the answer type. To learn how to configure questions for each answer type, see the subsections below.\u00a0 
\n
Free Form
\n
To configure free form questions, you'll only need to assign points to them.
\n
To assign points, enter a number of points into the Points field. Because free-form questions don't have correct answers, KCM GRC won't give a vendor points for their answer to the question. When you review the questionnaire, you can enter the number of points that you would like to give the vendor for their answer.
\n\n
Multiple Choice
\n
To configure multiple choice questions, you'll need to add answer options and assign points.\u00a0
\n
For more information, see the steps below:
\n
    \n
  1. Under Answer Options, enter two answer options.
    2. \n
  2. (Optional) Add additional answer options by clicking the + Add Option\u00a0button.
    3. \n
  3. Select a radio button under Correct to specify the correct answer.\n\n
    4. \n
  4. In the Points field, enter the number of points to assign to the question.\n
    \n
    \nTip: If you would like to assign the same number of points to all questions in the questionnaire, you can skip this step. After you've added all questions to the questionnaire, you can enter the point value into the Set For All field. Then, click the\u00a0Set For All button.
    \n
    \n
    5. \n
  5. Click the\u00a0Add\u00a0button to add the question to the questionnaire. \n\n
    6. \n
\n\n
Checkbox
\n
To configure checkbox questions, you'll need to add answer options and assign points.
\n
For more information, see the steps below:
\n
    \n
  1. Under Answer Options, enter two answer options.
    2. \n
  2. (Optional) Add additional answer options by clicking the + Add Option\u00a0button.
    3. \n
  3. Click the\u00a0Add\u00a0button to add the question to the questionnaire. \n\n
    4. \n
  4. Find the question in the\u00a0Questions list.
    5. \n
  5. Click the\u00a0pencil icon to edit the question.\u00a0\n\n
    6. \n
  6. In the Weight field, enter the weight to assign to each answer option. The total weight of the answer options will be the number of points assigned to the question.\n
    \n
    \nNote: As a best practice, we recommend that you assign a weight of 0 to the undesirable answer options.
    \n
    \n
    7. \n
  7. Click the\u00a0check mark icon to finish configuring the question.
    8. \n
\n
Yes / No / N/A
\n
To configure Yes / No / N/A questions, you'll need to select a correct answer and assign points.
\n
For more information, see the steps below:
\n
    \n
  1. In the Correct column, select the radio button next to the correct answer.\u00a0 \n\n
    2. \n
  2. In the Points field, enter the number of points to assign to the question.\n
    \n
    \nTip: If you would like to assign the same number of points to all questions in the questionnaire, you can skip this step. After you've added all questions to the questionnaire, you can enter the point value into the Set For All field. Then, click the\u00a0Set For All button.
    \n
    \n
    3. \n
  3. Click the\u00a0Add\u00a0button to add the question to the questionnaire. \n\n
    4. \n
\n\n
Creating a Custom Questionnaire Template
\n
Once your questionnaire is in Reviewed status, you can save the questionnaire as a custom questionnaire template. Then, you will be able to reuse the custom questionnaire template to create new questionnaires.\u00a0
\n
To create a custom questionnaire template, follow the steps below:
\n
    \n
  1. From the navigation panel, navigate to Vendor Management\u00a0>\u00a0Questionnaires.
    2. \n
  2. From the Name column, select the questionnaire that you would like to create a questionnaire template from.
    3. \n
  3. Click the Create Template button at the top-right corner of the page.\n\n
    4. \n
  4. In the confirmation pop-up window that opens, click the Yes button.
    5. \n
\n
After you complete the steps above, you can use the questionnaire template to create a new questionnaire. When you create a questionnaire, you can find your questionnaire template by clicking the list button in the\u00a0Add Questions From section of the page.
