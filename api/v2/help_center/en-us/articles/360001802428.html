In the ModStore, you can download video modules in MP4 or SCORM file format. Before you download video modules, we recommend that you add the modules to your library so you can track which videos you have downloaded. After you download a video module from the ModStore, your internet browser will either download the video module to your file manager or prompt you to save the video module to your file manager.
\n
\nImportant: If you are unable to find video module or training module files that are available for download, contact your Customer Success Manager. If you don't know who your Customer Success Manager is, contact our support team.
\n
See the section below to learn how to download a video module and how your internet browser will download the video module to your file manager. For general information about our ModStore content, see our ModStore and Library Guide.\u00a0
\n\n
Downloading Video Modules
\n
To download a video module from the ModStore, follow the steps below:
\n
    \n
  1. From the ModStore tab, hover your mouse over the video module that you would like to download.
    2. \n
  2. Click the View Details link.\u00a0 \n
    \nTip:If you already added the video module to your library, you can also access the video module from the Library subtab of the ModStore. From the Library subtab, click the information icon next to the module that you would like to download.\n\n
    \n\n
    3. \n
  3. Click the Download button.\u00a0
    4. \n
  4. From the Download Options pop-up window, click the Download link next to the video format you would like to download.\n\n
    5. \n
\n\n\n
Internet Browser Behavior
\n
When you click the\u00a0Download button, your internet browser will either automatically download the video module or prompt you to save the file to your device.\u00a0
\n
For information about the default settings for your browser, view the list below.
\n
    \n
  • \nFirefox: By default, Firefox will prompt you to save the file. If you would like to change the file name or location after you click the OK button, you can do so by locating the file in your file manager.\u00a0
    • \n
  • \nGoogle Chrome: By default, Google Chrome will prompt you to save the file. You can change the file name and location before you click the Save File radio button.\u00a0
    • \n
  • \nMicrosoft Edge: By default, Microsoft Edge will download the file to your Downloads folder. If you would like to change the file name or location, you can do so by locating the file in your file manager.\u00a0
    • \n
  • \nSafari: By default, Safari will download the file to your Downloads folder. If you would like to change the file name or location, you can do so by locating the file in your file manager.\u00a0
