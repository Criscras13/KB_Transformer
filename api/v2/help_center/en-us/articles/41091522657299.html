This article discusses the legacy COM add-in. We strongly encourage organizations to use the web add-in for a full, feature-rich experience as the COM add-in includes functionality not currently supported within the Microsoft Office JS framework. More information about the web add-in can be found in the Web Add-In Knowledge Base section.
\n
The legacy COM add-in integrates with Microsoft Outlook Desktop for Windows on desktop and laptop systems, enhancing email security and communication management. The add-in monitors outgoing messages to identify potentially misaddressed recipients and facilitates email encryption through the standard Microsoft Outlook interface. Users can read, compose, and respond to secure messages without leaving their email client. The add-in includes access management features allowing for message revocation when necessary, providing control over sensitive information after transmission.
