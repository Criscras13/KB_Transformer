Policies are used to determine the behavior of the Microsoft Outlook COM add-in. When changes are made to your policies, you must refresh the policies in your Microsoft Outlook COM add-in.
\n
To do this, follow the instructions below:
\n
    \n
  1. Open the File menu in Microsoft Outlook.
    2. \n
  2. Select Egress.
    3. \n
  3. Ensure you are signed in and select Refresh Policies.
    4. \n
  4. Once you've done this, restart Microsoft Outlook. You will then have the latest version of your organization's policies, including any changes or updates.
