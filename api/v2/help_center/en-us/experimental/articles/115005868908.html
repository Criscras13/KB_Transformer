There are a few ways to change or reset your users' passwords from the KSAT admin console. Click on the links below to see the steps for each process.
\n\n
Reset User Passwords Individually
\n
Follow the steps below to send a password reset link directly to the confirmed email address of the affected user:
\n
    \n
  1. Log out of your KnowBe4 admin account. To do this, click your email address in the top-right corner and select Log Out.
    2. \n
  2. In a new window or tab, open our KnowBe4's Training Instances article. Then, click on the password reset link for your training instance.\n\n
    3. \n
  3. In the Email field, enter the affected user's email address.\n
      \n
    • This must be the email address used for their KnowBe4 training.
      • \n
    \n
    4. \n
  4. Click Send me reset password instructions.The user should receive an email with a reset link within minutes. The link will be valid for approximately six hours.If they do not receive this email, we suggest checking their Junk or Spam folder. If they are unable to locate the email, we recommend trying the other options in this article or seeing our How to Manage Your Password article for more information.
    5. \n
\n
\nNote:Alternatively, you can send your users a direct link to the password reset link from our KnowBe4's Training Instances article.
\n\n\n
Assign User Passwords in Bulk Using Quick Import
\n
You can manually change the password for one or more users using the\u00a0Quick Import feature. This option allows you to set the same password for all of the users you choose to provide. To use this process, follow these steps:
\n
    \n
  1. From the KSAT console, navigate to Users > Import Users and click on\u00a0Quick Import.
    2. \n
  2. Enter the email address of each user you would like to change the password for.\u00a0\n
      \n
    • Only one email should be entered per line.
      • \n
    • Do not use any commas or spaces.
      • \n
    \n
    3. \n
  3. Select the Set Password for Users check box.
    4. \n
  4. Enter a password for these users.\n
      \n
    • The same password will be assigned to all users provided on this page.
      • \n
    \n
    5. \n
  5. Click Import Users to update these users' passwords.\n\n
    6. \n
\n\n\n
Assign User Passwords in Bulk Using CSV Import
\n
You can manually change the password for one or more users using the\u00a0CSV Import feature. This option allows you to set unique passwords for each user of your choice. To use this process, follow these steps:
\n
    \n
  1. Create a CSV file with two headers: Email and Password.
    2. \n
  2. Under the Email header, enter\u00a0the email address of each user whose password you would like to change.\n
      \n
    • If you need to change the password for all of your users, you can quickly obtain a list of all of your users from the Users tab of your KSAT console. Above the users list, click Generate CSV.\u00a0Be sure to remove yourself from this list though, or else your password will be changed as well.
      • \n
    \n
    3. \n
  3. Under the Password header, enter the desired password for that user.\u00a0I
    4. \n
  4. Save the CSV file.\n
    \nImportant:\nIf you are using Excel, CSV files must be saved in the CSV UTF-8 format.\n
    \n
    5. \n
  5. In the KSAT console, navigate to Users > Import Users and click on\u00a0CSV Import.
    6. \n
  6. Click the\u00a0Browse button and upload your CSV.\u00a0\n\n
    7. \n
  7. Choose your notification settings, then click\u00a0Import Users at the bottom of the page.The users you included in the CSV file will now be able to log in with the password you assigned.\u00a0
    8. \n
