You have the option to install our Hybrid Phish Alert Button (PAB) or Microsoft Ribbon Phish Alert Button (PAB) on a single machine with an Outlook on the Web (OWA) account.
To install the Hybrid PAB or Microsoft Ribbon PAB on a single machine with Outlook, follow the steps below:
    \n
  1. Log in to your OWA account.
    2. \n
  2. In your browser, navigate to https://aka.ms/olksideload.
    3. \n
  3. \n
    Outlook will open with an Add-Ins for Outlook window. Select My add-ins on the left side of the screen.
    \n
    \n
    4. \n
  4. Scroll down to the Custom Addins section.
    5. \n
  5. \n
    Select the + Add a custom add-in drop-down menu and select the Add from file... option.
    \n
    \n
    6. \n
  6. Choose the PhishAlertManifest.xml file for the Hybrid PAB or the PhishAlertManifestMSR.xml file for the Microsoft Ribbon PAB. You can download these files from your KnowBe4 Phishing Account Settings. Then, select the Open button.
    7. \n
  7. \n
    In the Warning pop-up window that opens, select Install.
    \n
    \n
    8. \n
  8. \n
    Once installed, close and reopen Outlook on the Web for the PAB to display in your account.
    \n
    \n
    9. \n
