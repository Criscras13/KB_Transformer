KnowBe4 provides several built-in SecurityTip notification templates that you can use when you create real-time coaching campaigns. You can also create your own custom notification templates. SecurityTips are localized based on the recipient's training language.
\n
For general information about SecurityCoach, see our SecurityCoach Product Manual.
\n
Creating a Custom SecurityTip Notification Template
\n
To learn how to create a custom SecurityTip notification template, follow the steps below:
\n
    \n
  1. Log in to your KSAT console and navigate to SecurityCoach > SecurityTips > Notification Templates.
    2. \n
  2. Click + Create Notification Template at the top-right corner of the page.
    3. \n
  3. Fill out the fields on the New SecurityCoach Notification page. For more information about these fields, see below:\n
    \n
      \n
    1. \nTemplate Name: Enter a name for your template.
      2. \n
    2. \nStatus: This drop-down menu defaults to Draft and cannot be changed until you save the template.
      3. \n
    \n
    4. \n
  4. Click Save. When you click, the Translations table will display.
    5. \n
  5. Click + Add Language to open the What You See Is What You Get (WYSIWYG) editor. For more information about the fields in this editor, see below:\n
    \n
      \n
    1. \nLanguage: Select a language for the translation.\u00a0
      2. \n
    2. \nSender\u2019s Email Address: Enter the email address that the SecurityTip email will appear to come from. You can use any email address you\u2019d like.
      3. \n
    3. \nSender\u2019s Name: Enter the name that will appear next to the Sender\u2019s Email Address when a user receives a SecurityTip by email.
      4. \n
    4. \nSubject: Enter a subject for the template. This subject will be visible to your users.
      5. \n
    5. \nBody: Enter any text that you\u2019d like to display in the body of the template.
      6. \n
    \n
    6. \n
  6. Click Save to create the translation for this template.
    7. \n
  7. Optionally, repeat steps 5 through step 6 to add additional translations for the template.
    8. \n
  8. From the Status drop-down menu, select Active. Notification templates must be active for you to use them in a real-time coaching campaign.
    9. \n
  9. Click Save.
    10. \n
\n
Managing and Editing SecurityTip Notification Templates
\n
Once you have created a custom SecurityTip notification template, you can manage or edit the template at any time. To manage and edit your templates, navigate to SecurityCoach > SecurityTips > Notification Templates. You can find your existing templates under the My Templates section of the page.
\n
\n
\nNote:If you have used the template in an active real-time coaching campaign, you cannot delete or archive the template.
\n
\n
To learn more about the options available in the My Templates section, see below:
\n
    \n
  1. \nMy Templates: In this section, you can view a table that lists your custom templates. For each template, you can see the Language Count, Date Updated, and Status.
    2. \n
  2. \nStatus: Select a status to filter templates by. You can choose from All, Active, Drafts, or Archived.
    3. \n
  3. \nLanguages: Click this drop-down menu to filter content by language.
    4. \n
  4. \nSearch: Search for a template by entering a keyword used in the template name.
    5. \n
  5. \nTemplate Name: Click a template name to edit that template.
    6. \n
  6. \nMore Options: Click this icon to open a drop-down menu with more options, including the Clone and Delete buttons.
    7. \n
  7. \nPreview: Click this icon to preview the template.
    8. \n
