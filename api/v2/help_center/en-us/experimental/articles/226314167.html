The Scam of the Week is a weekly message written by KnowBe4. The message summarizes a recent real-world scam and provides tips on how to stay safe from similar scams. To send the message to your users, you can use our user messaging feature to create and send a recurring message.
\n
If you want your users to be able to view any Scam of the Week sent to them, we encourage you to set it up as a message. Every message will remain listed in the KSAT admin console under Users > Messages and in the Messages tab of the Learner Experience (LX). When you send a Scam of the Week message to your users, the message will remain in the Messages tab of the LX until you hide it.
\n
\n
\nTip:If you want to send a message with a specific theme, we recommend using the\u00a0Security Hints and Tips\u00a0message. These messages cover similar themes to the Scam of the Week, but they avoid mentioning specific events.
\n
\n
Set Up a Scam of the Week Message
\n
Users will be able to view the Scam of the Week on the Messages tab of the Learner Experience (LX). You can customize the frequency you send these messages, and you have the option to send a copy of the message to each user via email. For general information about messages, see our User Messaging Guide.
\n
\n
\nNote:Scam of the Week messages are now available in 34 languages for better localization support. To access this new feature, admins must enable the Enable Localized Training Notifications toggle in Account Settings > Training > Learner Experience.
\n
\n

    4. \nMessage Delivery Date: In this field, select when you would like to send your message.
      \n
      \n
      \nNote:When using a frequency other than One-time, each subsequent message will be sent on the day of the week that was selected.

      \n

      \n
      \nTip:Emails sent using the User Messaging feature are not tracked and will not be included in any message-related data.
