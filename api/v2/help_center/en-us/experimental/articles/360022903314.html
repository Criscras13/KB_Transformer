\n
\nNote:We\u2019ve added advanced features to phishing templates! All accounts will be opted in to phishing template advanced features in 2025.\u00a0For more information, see our Phishing Templates Advanced Features Guide.
\n
\n
If you'd like to ensure you're using the latest Current Events or Holiday content for your end users, you can use our Current Event of the Week or Current Event of the Month phishing template topics in an ongoing campaign.
\n
Each topic will contain a single Current Events or Holiday template, hand-selected by the Content team each week and month to ensure that you're getting a high-quality, timely, and relevant template.
\n
Once a Current Event of the Week/Month campaign is set up, it will run automatically and will test your users with a new Current Events or Holiday phishing template each time the campaign runs.
\n
These templates may use links, attachments, or QR codes to test your users.
\n
\n
\nImportant:Cybercriminals often use shocking information or tactics to get users to fall for phishing attacks. Therefore, to help prepare your users, shocking techniques may also be used in these template topics. If your company culture requires that you get approval on the kinds of templates you use to test your users, you should\u00a0not set up an automated phishing campaign using these topics. Current Events templates may use scary headlines, sad stories, or other techniques that are meant to evoke an emotional response in your users and encourage them to click a link, open an attachment, or scan a QR code.
\n
\n
If you'd like to continue with setting up your campaign, follow the below recommendations.
\n
Recommended Settings for Your Current Event of the Week Campaign
\n
Campaign Name: Current Event of the Week
\n
Start Time:\u00a0We recommend setting this up for Wednesdays\u00a0or\u00a0Thursdays. The template is updated on Mondays.
\n
Deliver To: All Users
\n
Frequency: Weekly
\n
Sending:\u00a0Send all emails when the campaign starts
\n
Topics: Current Event of the Week
\n
*Do not select a specific template from the topic. Leave it on the default setting.
\n
The Current Event of the Week topic
\n
will be updated weekly on Mondays. Once this campaign is set up, your Current Event of the Week phishing campaign will run automatically each week, testing your users with a new, hand-selected Current Events or Holiday phishing template each time the campaign runs.
\n
Recommended Settings for Your Current Event of the Month Campaign
\n
Campaign Name: Current Event of the Month
\n
Start Time:\u00a0We recommend setting this up to run around the 15th of the month. The template is updated on the first Monday of each month.
\n
Deliver To: All Users
\n
Frequency: Monthly
\n
Sending:\u00a0Send all emails when the campaign starts
\n
Topics: Current Event of the Month
\n
*Do not select a specific template from the topic. Leave it on the default setting.
\n
The Current Event of the Month topic will be updated monthly on the first Monday of each month. Once this campaign is set up, your Current Event of the Month phishing campaign will run automatically each month, testing your users with a new, hand-selected Current Events or Holiday phishing template each time the campaign runs.
