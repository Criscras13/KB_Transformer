The KB4Diag tool is an executable file that KnowBe4's support team uses to automatically gather issue-specific files, including event and security logs and other files for troubleshooting. The support team requests that users run this tool when encountering issues with RanSim.
\n
\n
\nNote:When using the KB4Diag tool for RanSim, application logs need to be added separately. To find the application logs, please locate the folder where the product is installed. For more information, see our RanSim Product Manual.
\n
\n
After you download and run KB4Diag, the tool will gather all relevant files and package the files into a zip file. Then, you can send these files to the support technician from your ticket so that the support technician can view the files.\u00a0
\n
\n
\nNote: If you prefer not to use KB4Diag to locate and gather these files, you can ask our support team which files they need to troubleshoot your issue. Then, you can locate these files manually.\u00a0
\n
\n
Installing and Running KB4Diag
\n
After a support technician requests that you run the KB4Diag tool, please follow the instructions below.\u00a0
\n
    \n
  1. Download the executable file from the link that the support technician included in your ticket.\n
    \n
    \nImportant: You must download the file onto the machine where the affected program is installed.
    \n
    \n
    2. \n
  2. Navigate to the KB4Diag file in the Downloads folder of your File Explorer.
    3. \n
  3. Double-click the file to run it.\u00a0 \n
    4. \n
  4. Click Run in the pop-up window. \n
    5. \n
  5. Check the box to agree to the End User License Agreement, and click Continue. \n
    6. \n
  6. After the tool runs, please follow the instructions below.\n
      \n
    1. \nRun:\u00a0Click this button to run KB4Diag.
      2. \n
    2. \nBrief Description of Issue: Add details about the issue you are encountering with the product.
      3. \n
    3. \nTicket Product Type: Select RanSim from the drop-down menu.
      4. \n
    4. \nTicket Email Address: Enter your preferred email address. This email address does not need to match the email address that you used to submit the support request.\u00a0
      5. \n
    5. \nTicket ID: Enter the ticket number from your support ticket.
      6. \n
    \n
    7. \n
  7. Wait for the scan to complete. The scan can take up to 10 minutes or longer depending on the number of files that the tool gathers.\n
    8. \n
  8. Locate any files that you want to include in your support ticket by following the steps below.\u00a0\n
      \n
    1. Open your File Explorer.
      2. \n
    2. Navigate to ProgramData\\KnowBe4\\[[Affected Product]]\\Support.\u00a0\n
      3. \n
    3. Review the files in the folder above.
      4. \n
    4. Upload any necessary files to your support request.\u00a0If you are responding to your support request via email, attach any files to the email. \n
      5. \n
    \n
    9. \n
\n
Files That KB4Diag Gathers
\n
When you run KB4Diag, the tool gathers the files below:\u00a0
\n
    \n
  • Application logs for the affected program
    • \n
  • Configuration files for the affected program
    • \n
  • Registry keys for the affected program
    • \n
  • Last five days of Application Event logs
    • \n
  • Last five days of System Event logs
    • \n
  • Last three days of Security Event logs
    • \n
  • MSInfo ipconfig.txt
    • \n
  • Files in InstallDir for the affected program
    • \n
  • List of all currently running processes
    • \n
  • Data entered into Diagnostics UI
    • \n
\n
Troubleshooting KB4Diag
\n
When you run KB4Diag, you may encounter the error below:
\n
If you encounter this error, please follow the steps below:
\n
    \n
  1. Save the KB4Diag file to your Downloads folder.\u00a0
    2. \n
  2. Run the file from its saved location.\u00a0
    3. \n
\n
If the steps above do not resolve the error, or if you encounter a different error, please contact the KnowBe4 Support Team.\u00a0
