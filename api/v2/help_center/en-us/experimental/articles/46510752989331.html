You can integrate your Tray platform into your KSAT console. Tray allows you to make highly customizable and easy-to-build workflows, enabling deep integrations with niche, proprietary, or highly customized internal security tools. Tray utilizes KnowBe4\u2019s Graph APIs to enable automated security awareness training workflows and data management in areas including groups, users, phishing, and training.
\n
Before You Connect Tray to Your KSAT Console
\n
Before integrating Tray into your KSAT console, you must find your API address and create a Product API token. Follow the steps below to do both actions.
\n
\n
\nNote:KnowBe4's Graph APIs are available to Diamond-level customers. For more information about Graph APIs, please see our KnowBe4 Graph API Overview article.
\n
\n
    \n
  1. Your API address depends on where your KnowBe4 account is located. Please see the table below to find the API address for your region.\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n
    RegionBase URL
    UShttps://us.api.knowbe4.com
    CAhttps://ca.api.knowbe4.com
    EUhttps://eu.api.knowbe4.com
    UKhttps://uk.api.knowbe4.com
    DEhttps://de.api.knowbe4.com
    \n
    2. \n
  2. Log in to your KSAT console as an admin.
    3. \n
  3. Navigate to Account Settings in the top-right corner under your account\u2019s email address.
    4. \n
  4. In Account Settings, navigate to Account Integrations.
    5. \n
  5. Under Product API, select Product API. The Product API subtab will open.
    6. \n
  6. At the top-right corner of the Product API subtab, select + Create New API Token.
    7. \n
  7. Fill out the fields on the Create New API Token page.
    8. \n
  8. Select Create Token. The Product API Token pop-up window will open.
    9. \n
  9. Select the Product API token to copy the token. Be sure to save your copied API token in a location that you can easily access. After you close this window, you will not be able to view this token again.
    10. \n
\n
Connecting Tray to Your KSAT Console
\n
You can connect your Tray platform to your KSAT console by following the steps below.
\n
    \n
  1. Log in to your Tray platform as an admin.
    2. \n
  2. Navigate to your desired workspace for testing your connector.
    3. \n
  3. Navigate to the Services tab.
    4. \n
  4. Select the + New service button to create a new service for KnowBe4. In this example, KnowBe4 Test Service is used.\n
      
    \n
      
    \n
    5. \n
  5. In Edit service, add required authentication parameters, including access_token and select_region, as shown below.\n
      
    \n
      
    \n
    6. \n
  6. Save the service created by selecting Save in the upper-right corner of the page.
    7. \n
  7. Navigate to the Authentication tab.
    8. \n
  8. Select + Create new authentication. In the drop-down menu, select From scratch.\n
      
    \n
    9. \n
  9. From the pop-up window, select the KnowBe4 service you created and provide other required details, such as the authentication name.\n
      
    \n
    10. \n
  10. Enter the Product API token and API address you received in the previous steps.\n
      
    \n
    11. \n
\n
Setting Up KnowBe4 Workflows in Tray
\n
You can set up KSAT workflows in your Tray platform by following the steps below.
\n
    \n
  1. Navigate to the Projects tab.
    2. \n
  2. Navigate to Add project > New project.\n
      
    \n
    3. \n
  3. Add your project name (required), description, and tags to your project. When finished, select Create.\n
      
    \n
    4. \n
  4. In the project you created, navigate to + Add workflow > From scratch.\n
      
    \n
    5. \n
  5. In the new workflow, name your workflow. This field is required.\n
      
    \n
    6. \n
  6. On the next page, select a Manual trigger. After selecting the trigger, select Create workflow.\n
      
    \n
    7. \n
  7. Select the KnowBe4 connector.\n
      
    \n
    8. \n
  8. When you select the KnowBe4 connector, 37 operations should appear in a drop-down menu. For this example, we will test the Add User to Training Campaign operation.\n
      
    \n
    9. \n
  9. Select Authentication to begin running the operation.\n
      
    \n
    10. \n
  10. Provide Inputs and select Run step to run a single operation. To run the entire workflow, select Run workflow.\n
      
    \n
    11. \n
  11. You can enable the Show logs toggle at the top-right corner of the page to view input and output.\n
      
    \n
    12. \n
\n
If you experience issues integrating Tray with your KSAT console, please contact Tray\u2019s technical support.
