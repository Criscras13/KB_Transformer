When you download a CSV file from your KSAT console, you can open the file in a spreadsheet program such as Microsoft Excel. However, if the CSV file contains special characters that are not included in basic Latin script, these special characters may not display in Microsoft Excel.
\n
To ensure that special characters display in your CSV file in Microsoft Excel, follow the steps below:
\n
    \n
  1. Open a blank workbook in Microsoft Excel.
    2. \n
  2. Under the Data tab, navigate to Get External Data > From Text.\n
    3. \n
  3. In the pop-up window that opens, select the CSV file you downloaded.
    4. \n
  4. Click Get Data.
    5. \n
  5. From the File origin: drop-down menu, select Unicode (UTF-8).\n
    \n
    \nNote: Ensure that Delimited is selected for the Data Type. Selecting the Delimited option will ensure that data is separated into columns.
    \n
    \n\n
    6. \n
  6. Click Next >.
    7. \n
  7. Under Delimiters, select the Comma check box. \n
    8. \n
  8. Click Next >.
    9. \n
  9. On the final screen, you will have the option to set the data format for each column. You can leave this setting as the default General setting. \n
    10. \n
  10. Click Finish.
    11. \n
  11. In the Import Data pop-up window, select the sheet where you want the data to be imported to and click OK. \n
    12. \n
