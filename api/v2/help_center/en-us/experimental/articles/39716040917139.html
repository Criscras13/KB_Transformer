Microsoft has released the newest version of Microsoft Outlook: New Outlook for Windows. This version is generally available to commercial users, and users can opt-in to try the new experience:
\n
  
\n
Admins should be aware that COM add-ins are not supported in the New Outlook for Windows. As a result, you will not be able to use our COM add-in. For further information, see Microsoft's Outlook Add-Ins Overview article.
\n
There are two options admins can take. These are:
\n
    \n
  1. \nHide the \"Try the new Outlook\" button. If you do not want users to be able to install the new Outlook, hide the button using the following registry key:\n
      \n
    1. HKEY_CURRENT_USER\\Software\\Microsoft\\Office\\16.0\\Outlook\\Options\\General
      2. \n
    2. DWORD with the name HideNewOutlookToggle and its value set to 1.
      3. \n
    \n
    2. \n
  2. \nDownload our Web add-in. The Web add-in is fully supported if you want to use the new Outlook.
