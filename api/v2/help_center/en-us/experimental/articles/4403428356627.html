In the KCM GRC Vendor Risk Management module, you can send questionnaire assessments to your organization's vendors or third parties. Then, you can use the Send Link button to send a follow-up email that prompts vendors to complete questionnaires. For more details about the full vendor workflow, see our Vendor Risk Management Module Guide.\u00a0
\n
You can find the\u00a0Send Link\u00a0button by selecting a vendor profile on the Vendor Details\u00a0page (Vendor Management >\u00a0Vendors) and navigating to the Assigned Questionnaires tab.
\n
The Send Link\u00a0button creates an email draft that contains a link to the questionnaire. This email draft automatically opens in your device's default email application. If your default email application is not your preferred email application, this article can help you change your default settings in web browsers, Windows 10 OS, and Mac OS.
\n
\n
Change Your Default Email Web Application for Your Web Browser
\n
The web browsers Chrome, Edge, and Firefox automatically open email links in your device's default email application, which is typically a desktop application. If you want the Send Link\u00a0button to open a web application instead of a desktop application, you can change the default email web application for any browser without affecting your device's default settings.
\n
To learn how to change your default email settings, select your browser from the tabs below.
\n
\n
\n
Change Default Email Web Application in Google Chrome
\n
To change your default email web application in Chrome, follow the instructions below.\u00a0
\n
    \n
  1. Open Google Chrome.
    2. \n
  2. Click the three vertical dots icon in the top-right corner.\u00a0
    3. \n
  3. Click Settings.\n
    4. \n
  4. Select\u00a0Privacy and security\u00a0from the navigation bar.\u00a0
    5. \n
  5. Select\u00a0Site Settings\u00a0under\u00a0Privacy and security.
    6. \n
  6. Click Additional permissions under\u00a0Permissions.
    7. \n
  7. Select\u00a0Handlers.
    8. \n
  8. Select\u00a0Sites can ask to handle protocols.\u00a0If this was already selected, make sure that your preferred email web application is not blocked.
    9. \n
  9. Navigate to your preferred email web application. For example, navigate to emailclient.com.
    10. \n
  10. Click the double-diamond icon in your address bar.\u00a0
    11. \n
  11. Select\u00a0Allow\u00a0under Allow [web application] to open all email links?. \n
    12. \n

    \n
  1. Open Microsoft Edge.\u00a0

    3. \n

    4. \n
  4. Select\u00a0Handlers\u00a0from the list of site permissions.
    5. \n
  5. Select\u00a0Sites can ask to handle protocols.\u00a0If this was already selected, make sure that your preferred email web application is not blocked.
    6. \n
  6. Navigate to your preferred email web application. For example, navigate to emailclient.com.
    7. \n
  7. Click the double-diamond icon in your address bar.\u00a0
    8. \n

Change Default Email Web Application in Firefox
\n
To change your default email web application in Firefox, follow the instructions below.\u00a0
\n
    \n
  1. Open Firefox.
    2. \n
  2. Click the three horizontal lines icon in the top-right corner.
    3. \n
  3. Select Preferences if you're using Mac or Options if you're using\u00a0Windows.
    4. \n
  4. Navigate to General >\u00a0Applications.
    5. \n
  5. Enter \"mailto\" into the search bar.
    6. \n
  6. Select mailto from the Content Type\u00a0section.
    7. \n
  7. Click the drop-down arrow next to\u00a0Action.
    8. \n
  8. Select your preferred email web application from the list of applications.\u00a0
    9. \n

\n

    4. \n
  4. Select\u00a0Default Apps\u00a0from the navigation bar.
    5. \n
  5. Click the current default email application under Email.\u00a0
    6. \n
  6. Select your preferred email application from the list of applications.\u00a0\n
